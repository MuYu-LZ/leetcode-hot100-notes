# 240. 搜索二维矩阵 II

## 题目
编写一个高效的算法来搜索 m x n 矩阵 matrix 中的一个目标值 target。该矩阵具有以下特性：

整数在每一行按从左到右递增。
整数在每一列按从上到下递增。

示例 1:
输入: matrix = [[1,4,7,11],[2,5,8,12],[3,6,9,16],[10,13,14,17]], target = 5
输出: true

示例 2:
输入: matrix = [[1,4,7,11],[2,5,8,12],[3,6,9,16],[10,13,14,17]], target = 13
输出: false

## 思路
该题利用矩阵的**递增特性**，采用**从右上角或左下角开始搜索**的策略：
1. 从矩阵的右上角（或左下角）开始
2. 当前值与目标值比较：
   - 如果相等，返回 true
   - 如果当前值大于目标值，向左移动（或向上移动）
   - 如果当前值小于目标值，向下移动（或向右移动）
3. 直到搜索到目标值或走出矩阵边界

关键思想：从某个"拐角"出发，可以利用矩阵的单调性，将二维搜索转化为一次有效的遍历。

## Python 代码

```python
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        # 1. 边界处理
        if not matrix or not matrix[0]:
            return False
        
        # 2. 从右上角开始（也可从左下角开始）
        row = 0
        col = len(matrix[0]) - 1
        
        # 3. 进行搜索
        while row < len(matrix) and col >= 0:
            if matrix[row][col] == target:
                # 4. 找到目标值
                return True
            elif matrix[row][col] > target:
                # 5. 当前值过大，向左移动
                col -= 1
            else:
                # 6. 当前值过小，向下移动
                row += 1
        
        # 7. 未找到目标值
        return False

```

## 易错点
1. **起始位置的选择**：只能从右上角或左下角开始，不能从左上角或右下角开始
   - 从右上角开始：大的值在右，小的值在左；大的值在下，小的值在上
   - 从左下角开始：大的值在下，小的值在上；大的值在右，小的值在左
2. **移动方向**：从右上角出发，要么左移（减少列），要么下移（增加行）
3. **边界检查**：循环条件是 row < len(matrix) and col >= 0
4. **时间复杂度**：O(m + n)，其中 m 是矩阵的行数，n 是矩阵的列数
