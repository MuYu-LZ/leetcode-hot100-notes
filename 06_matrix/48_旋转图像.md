# 048. 旋转图像

## 题目
给定一个 n × n 的二维矩阵 matrix，将其顺时针旋转 90 度。

必须在原地旋转图像，即不能申请额外的二维数组来进行旋转。

示例 1:
输入: matrix = [[1,2,3],[4,5,6],[7,8,9]]
输出: [[7,4,1],[8,5,2],[9,6,3]]

示例 2:
输入: matrix = [[1]]
输出: [[1]]

示例 3:
输入: matrix = [[1,2],[3,4]]
输出: [[3,1],[2,4]]

## 思路
该题使用**转置 + 反转**的两步方法：
1. 第一步：沿主对角线进行转置
   - 交换 matrix[i][j] 和 matrix[j][i]
2. 第二步：沿竖直中轴线反转每一行
   - 交换 matrix[i][0] 和 matrix[i][n-1]，以此类推

这样就实现了 90 度顺时针旋转。

原理演示（3x3 矩阵）：
- 原矩阵：[[1,2,3],[4,5,6],[7,8,9]]
- 转置后：[[1,4,7],[2,5,8],[3,6,9]]
- 反转行后：[[7,4,1],[8,5,2],[9,6,3]] ✓

## Python 代码

```python
class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        n = len(matrix)
        
        # 1. 第一步：沿主对角线转置
        for i in range(n):
            for j in range(i + 1, n):
                # 2. 交换 matrix[i][j] 和 matrix[j][i]
                matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
        
        # 3. 第二步：沿竖直中轴线反转每一行
        for i in range(n):
            # 4. 反转第 i 行
            matrix[i].reverse()

```

## 易错点
1. **转置的范围**：只需遍历上三角部分（j 从 i+1 开始），避免重复交换
2. **反转方式**：可以使用 reverse() 方法，或使用双指针手动反转
3. **原地操作**：整个过程不需要额外的矩阵，只进行原地交换
4. **时间空间复杂度**：时间 O(n²)（需要访问所有元素），空间 O(1)（不计算输入矩阵的空间）
