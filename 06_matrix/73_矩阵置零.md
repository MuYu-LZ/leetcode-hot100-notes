# 073. 矩阵置零

## 题目
给定一个 m x n 的矩阵，如果一个元素为 0 ，则将其所在行与列的所有元素都设为 0 。请在原地完成。

进阶：
- 一个直观的解决方案是使用 O(mn) 的额外空间，但这并不是一个好的解决方案。
- 一个简单的改进方案是使用 O(m + n) 的额外空间，但这仍然不是最优的。
- 你能想到一个仅使用常数空间的解决方案吗？

示例 1:
输入: matrix = [[1,1,1],[1,0,1],[1,1,1]]
输出: [[1,0,1],[0,0,0],[1,0,1]]

示例 2:
输入: matrix = [[0,1,2],[3,4,5],[1,3,0]]
输出: [[0,0,0],[0,4,0],[0,0,0]]

## 思路
该题使用**第一行和第一列作为标记**的技巧，实现 O(1) 额外空间：
1. 使用两个标志变量记录第一行和第一列是否需要被置零
2. 遍历矩阵，如果 matrix[i][j] == 0，则标记 matrix[i][0] 和 matrix[0][j] 为 0
3. 根据第一行和第一列的标记，将对应的行列置零
4. 最后根据两个标志变量，单独处理第一行和第一列

## Python 代码

```python
class Solution:
    def setZeroes(self, matrix: List[List[int]]) -> None:
        # 1. 获取矩阵的行数和列数
        m, n = len(matrix), len(matrix[0])
        
        # 2. 标志变量：记录第一行和第一列是否需要置零
        first_row_zero = False
        first_col_zero = False
        
        # 3. 检查第一行是否有 0
        for j in range(n):
            if matrix[0][j] == 0:
                first_row_zero = True
                break
        
        # 4. 检查第一列是否有 0
        for i in range(m):
            if matrix[i][0] == 0:
                first_col_zero = True
                break
        
        # 5. 遍历矩阵（除了第一行和第一列）
        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][j] == 0:
                    # 6. 标记该行和该列
                    matrix[i][0] = 0
                    matrix[0][j] = 0
        
        # 7. 根据标记置零
        for i in range(1, m):
            for j in range(1, n):
                if matrix[i][0] == 0 or matrix[0][j] == 0:
                    matrix[i][j] = 0
        
        # 8. 根据标志变量处理第一行
        if first_row_zero:
            for j in range(n):
                matrix[0][j] = 0
        
        # 9. 根据标志变量处理第一列
        if first_col_zero:
            for i in range(m):
                matrix[i][0] = 0

```

## 易错点
1. **第一行和第一列的特殊处理**：必须先记录它们原来是否有 0，再用它们作为标记
2. **遍历范围**：在标记阶段，从 (1,1) 开始遍历，避免覆盖第一行和第一列的原始信息
3. **置零的顺序**：先根据标记置零中间部分，再单独处理第一行和第一列
4. **时间复杂度**：O(m × n)；空间复杂度：O(1)（只使用两个标志变量）
