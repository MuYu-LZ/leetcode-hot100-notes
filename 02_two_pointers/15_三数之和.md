# 015. 三数之和

## 题目
给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。

请你返回所有和为 0 的三元组。

注意：答案中不可以包含重复的三元组。

示例 1：
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
解释：
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 。
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 。
nums[0] + nums[4] + nums[5] = (-1) + (-1) + (-4) = 0 。
不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。

## 思路
该题使用**排序 + 双指针**的策略：
1. 首先对数组进行排序，便于去重和使用双指针
2. 遍历数组，对每个元素作为第一个数 nums[i]，在其右侧寻找两个数使得三数之和为 0
3. 对于每个 nums[i]，使用双指针（left 和 right）在 nums[i] 右侧寻找满足条件的两个数
   - 如果 nums[i] + nums[left] + nums[right] == 0，将其加入结果，并去重
   - 如果和小于 0，左指针右移
   - 如果和大于 0，右指针左移
4. 在遍历过程中去除重复的三元组

## Python 代码

```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        # 1. 排序数组
        nums.sort()
        result = []
        n = len(nums)
        
        # 2. 遍历数组，固定第一个数
        for i in range(n - 2):
            # 3. 如果第一个数已经大于 0，不可能有三数之和为 0
            if nums[i] > 0:
                break
            
            # 4. 去除重复的第一个数
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            
            # 5. 双指针寻找另外两个数
            left, right = i + 1, n - 1
            while left < right:
                total = nums[i] + nums[left] + nums[right]
                
                if total == 0:
                    result.append([nums[i], nums[left], nums[right]])
                    
                    # 6. 去除重复的左指针
                    while left < right and nums[left] == nums[left + 1]:
                        left += 1
                    # 7. 去除重复的右指针
                    while left < right and nums[right] == nums[right - 1]:
                        right -= 1
                    
                    # 8. 继续搜索其他解
                    left += 1
                    right -= 1
                elif total < 0:
                    left += 1
                else:
                    right -= 1
        
        return result

```

## 易错点
1. **排序的必要性**：排序不仅便于使用双指针，也为去重提供了便利（相同的元素相邻）
2. **去重策略**：需要在三个维度上去重：固定数、左指针和右指针。使用 while 循环跳过重复元素
3. **双指针移动**：找到解后，两个指针都要移动，继续寻找其他可能的三元组
4. **边界检查**：循环应遍历到 n-2，保证至少有两个元素留给双指针
