# 283. 移动零

## 题目
给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

必须在原数组上操作，不能拷贝额外的数组。

示例 1：
输入: nums = [0,1,0,3,12]
输出: [1,3,12,0,0]

示例 2：
输入: nums = [0]
输出: [0]

## 思路
该题使用**双指针**策略，利用"交换"的思想：
1. 使用两个指针：left 记录非零元素应该放置的位置，right 遍历整个数组
2. 当 right 指针遇到非零元素时，与 left 位置的元素交换，然后 left 右移
3. 当 right 指针遇到零时，仅 right 右移，不进行交换
4. 这样所有非零元素被按序移到前面，零自动到了末尾

关键思想：left 指针始终指向下一个应该放置非零元素的位置，right 指针负责寻找非零元素。

## Python 代码

```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        # 1. 初始化左指针，指向应该放置非零元素的位置
        left = 0
        
        # 2. 右指针遍历整个数组
        for right in range(len(nums)):
            # 3. 如果当前元素非零
            if nums[right] != 0:
                # 4. 将非零元素与 left 位置交换
                nums[left], nums[right] = nums[right], nums[left]
                # 5. left 右移，指向下一个应该放置非零元素的位置
                left += 1

```

## 易错点
1. **原地操作**：不能创建新数组，必须在原数组上修改
2. **保持相对顺序**：通过交换而非删除和插入来保持非零元素的原有顺序
3. **left 指针的含义**：left 始终指向"下一个非零元素应该放置的位置"，而不是"第一个零的位置"
4. **简洁性**：这个解法比找零的位置再移动更直接，时间复杂度为 O(n)，空间复杂度为 O(1)
