# 042. 接雨水

## 题目
给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接住多少雨水。

示例 1：
输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出：6
解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。

示例 2：
输入：height = [4,2,0,3,2,5]
输出：9

## 思路
该题使用**双指针**策略，利用装水量由左右两端较小值决定的原理：
1. 初始化左右两个指针指向数组两端，以及左右的最大高度
2. 每次移动较小一侧的指针：
   - 如果当前高度小于该侧的最大高度，则可以接水，水量 = 最大高度 - 当前高度
   - 如果当前高度大于等于该侧的最大高度，则更新最大高度
3. 重复步骤 2 直到左右指针相遇

关键思想：某位置的装水量取决于其左边的最大值和右边的最大值中的较小值。通过双指针和动态维护左右最大值，可以一次遍历求解。

## Python 代码

```python
class Solution:
    def trap(self, height: List[int]) -> int:
        # 1. 初始化双指针和最大高度
        left, right = 0, len(height) - 1
        left_max, right_max = 0, 0
        water = 0
        
        # 2. 双指针遍历
        while left < right:
            # 3. 移动较小一侧的指针
            if height[left] < height[right]:
                # 4. 处理左侧
                if height[left] >= left_max:
                    # 当前高度大于等于左侧最大值，更新最大值
                    left_max = height[left]
                else:
                    # 当前高度小于左侧最大值，可以接水
                    water += left_max - height[left]
                left += 1
            else:
                # 5. 处理右侧
                if height[right] >= right_max:
                    # 当前高度大于等于右侧最大值，更新最大值
                    right_max = height[right]
                else:
                    # 当前高度小于右侧最大值，可以接水
                    water += right_max - height[right]
                right -= 1
        
        return water

```

## 易错点
1. **移动指针的策略**：必须移动较小一侧的指针。因为装水量由 min(left_max, right_max) 决定，移动较小侧可以逐步确定装水量
2. **最大值的维护**：left_max 和 right_max 必须动态更新，表示当前位置左侧和右侧的最大高度
3. **边界条件**：当 height[left] == height[right] 时，移动哪一边都可以（代码选择移动右侧）
4. **时间空间复杂度**：O(n) 时间，O(1) 额外空间，相比动态规划方案更优
