# 011. 盛最多水的容器

## 题目
给定一个长度为 n 的整数数组 height ，其中有 n 条垂直的线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

返回容器可以容纳的最大水量。

说明：你不能倾斜容器。

示例 1：
输入：height = [1,8,6,2,5,4,8,3,7]
输出：49
解释：图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，最大面积是 49。

## 思路
该题使用**双指针**贪心法解决：
1. 初始化左右两个指针，分别指向数组的两端
2. 计算当前两条线围成的容器面积：面积 = min(height[left], height[right]) × (right - left)
3. 每次移动**较小的线所在的指针**（因为较小的线是面积的限制因素，移动它才可能增加面积）
4. 重复步骤 2-3 直到左右指针相遇，记录过程中的最大面积

关键思想：移动较长的那条线没有意义，因为新的高度最多等于那条线的高度（而实际由更短的线决定），同时宽度减小了，面积肯定不会增大。

## Python 代码

```python
class Solution:
    def maxArea(self, height: List[int]) -> int:
        # 1. 初始化双指针
        left, right = 0, len(height) - 1
        max_area = 0
        
        # 2. 双指针遍历
        while left < right:
            # 3. 计算当前容器的面积（宽度 × 最小高度）
            current_area = min(height[left], height[right]) * (right - left)
            # 4. 更新最大面积
            max_area = max(max_area, current_area)
            
            # 5. 移动较小的指针
            if height[left] < height[right]:
                left += 1
            else:
                right -= 1
        
        return max_area

```

## 易错点
1. **移动指针的判断**：必须移动较小高度的指针，而不是随意移动。这是贪心策略的核心
2. **面积计算公式**：面积由两条线中较短的那条限制，而宽度是指针之间的距离
3. **边界情况**：当 left == right 时停止循环，此时已经遍历了所有可能的配对
