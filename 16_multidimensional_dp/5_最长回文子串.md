# 5. 最长回文子串

## 题目
给你一个字符串 s，找到 s 中最长的回文子串。

示例:
- 输入: s = "babad"
  输出: "bab" 或 "aba"
  
- 输入: s = "cbbd"
  输出: "bb"
  
- 输入: s = "a"
  输出: "a"

## 思路
**方法一：动态规划**（O(n²)）：
1. dp[i][j] 表示 s[i:j+1] 是否为回文子串
2. 递推关系：
   - 如果 i == j，则 dp[i][j] = true（单个字符是回文）
   - 如果 i + 1 == j 且 s[i] == s[j]，则 dp[i][j] = true
   - 否则 dp[i][j] = dp[i+1][j-1] and s[i] == s[j]
3. 遍历顺序：从长度短到长，或者按 i 从大到小、j 从小到大
4. 追踪最长的回文子串

**方法二：中心扩展**（O(n²)）- 推荐：
对每个可能的中心（单个字符或两个字符之间），向两侧扩展，直到不再是回文为止。

关键：利用回文的对称性，减少状态空间

## Python 代码

```python
# 方法一：动态规划
def longestPalindrome(s):
    # 1. 特殊情况
    if not s:
        return ""
    
    n = len(s)
    # 2. dp[i][j] = s[i:j+1] 是否为回文
    dp = [[False] * n for _ in range(n)]
    start = 0
    max_len = 1
    
    # 3. 初始化单个字符为回文
    for i in range(n):
        dp[i][i] = True
    
    # 4. 从长度为 2 的子串开始
    for length in range(2, n + 1):
        for i in range(n - length + 1):
            j = i + length - 1
            
            if s[i] == s[j]:
                # 5. 长度为 2 的子串
                if length == 2:
                    dp[i][j] = True
                else:
                    # 6. 长度大于 2，检查内部是否为回文
                    dp[i][j] = dp[i + 1][j - 1]
                
                # 7. 更新最长回文子串
                if dp[i][j]:
                    start = i
                    max_len = length
    
    return s[start:start + max_len]

```

**方法二：中心扩展（更优雅）**：
```python
def longestPalindrome_expand(s):
    # 1. 定义中心扩展函数
    def expandAroundCenter(left, right):
        while left >= 0 and right < len(s) and s[left] == s[right]:
            left -= 1
            right += 1
        # 2. 返回回文子串
        return s[left + 1:right]
    
    # 3. 特殊情况
    if not s:
        return ""
    
    longest = ""
    
    # 4. 对每个可能的中心进行扩展
    for i in range(len(s)):
        # 5. 奇数长度的回文（单个字符为中心）
        p1 = expandAroundCenter(i, i)
        # 6. 偶数长度的回文（两个字符之间为中心）
        p2 = expandAroundCenter(i, i + 1)
        
        # 7. 选择较长的回文
        p = p1 if len(p1) > len(p2) else p2
        
        # 8. 更新最长回文
        if len(p) > len(longest):
            longest = p
    
    return longest
```

## 易错点
1. **DP 的递推关系**：需要检查 s[i] == s[j] 和内部是否为回文
2. **初始化**：单个字符总是回文，两个相同字符也是回文
3. **遍历顺序**：从短的子串到长的，保证依赖的子问题先被解决
4. **中心扩展的终止条件**：当字符不相等时停止扩展
5. **子串的提取**：s[left+1:right] 是正确的（left 和 right 已越界）
