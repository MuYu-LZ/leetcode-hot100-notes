# 64. 最小路径和

## 题目
给定一个包含非负整数的 m x n 网格 grid ，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

说明：每次只能向下或者向右移动一步。

示例:
- 输入: grid = [[1,3,1],[1,5,1],[4,1,1]]
  输出: 7
  解释: 路径 1 → 3 → 1 → 1 → 1 最小，总和为 7
  
- 输入: grid = [[1,2,3],[4,5,6]]
  输出: 12
  
- 输入: grid = [[1]]
  输出: 1

## 思路
使用**二维动态规划**：
1. dp[i][j] 表示到达位置 (i, j) 的最小路径和
2. 递推关系：dp[i][j] = grid[i][j] + min(dp[i-1][j], dp[i][j-1])
3. 边界处理：
   - 第一个元素：dp[0][0] = grid[0][0]
   - 第一行：只能从左边来，dp[0][j] = dp[0][j-1] + grid[0][j]
   - 第一列：只能从上边来，dp[i][0] = dp[i-1][0] + grid[i][0]
4. 空间优化：可以用一维数组或原地修改
5. 时间复杂度：O(m*n)

关键：利用最优子结构，每个位置的最小路径和取决于上方或左方的最小路径和

## Python 代码

```python
def minPathSum(grid):
    # 1. 初始化行数和列数
    m, n = len(grid), len(grid[0])
    
    # 2. 初始化 dp 数组
    dp = [[0] * n for _ in range(m)]
    
    # 3. 初始化第一个元素
    dp[0][0] = grid[0][0]
    
    # 4. 初始化第一列（只能从上边来）
    for i in range(1, m):
        dp[i][0] = dp[i - 1][0] + grid[i][0]
    
    # 5. 初始化第一行（只能从左边来）
    for j in range(1, n):
        dp[0][j] = dp[0][j - 1] + grid[0][j]
    
    # 6. 填充 dp 数组
    for i in range(1, m):
        for j in range(1, n):
            # 7. 从上方或左方选择最小的路径
            dp[i][j] = grid[i][j] + min(dp[i - 1][j], dp[i][j - 1])
    
    return dp[m - 1][n - 1]

```

**空间优化版本**（原地修改）：
```python
def minPathSum_optimize(grid):
    # 1. 直接修改原始网格
    m, n = len(grid), len(grid[0])
    
    # 2. 初始化第一列
    for i in range(1, m):
        grid[i][0] += grid[i - 1][0]
    
    # 3. 初始化第一行
    for j in range(1, n):
        grid[0][j] += grid[0][j - 1]
    
    # 4. 填充其余位置
    for i in range(1, m):
        for j in range(1, n):
            grid[i][j] += min(grid[i - 1][j], grid[i][j - 1])
    
    return grid[m - 1][n - 1]
```

## 易错点
1. **递推关系**：dp[i][j] = grid[i][j] + min(dp[i-1][j], dp[i][j-1])
2. **边界初始化**：第一个元素、第一行、第一列需要单独处理
3. **第一行初始化**：只能从左边来，累加所有元素
4. **第一列初始化**：只能从上边来，累加所有元素
5. **返回值**：dp[m-1][n-1]，即右下角的最小路径和
