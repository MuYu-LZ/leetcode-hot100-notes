# 62. 不同路径

## 题目
一个机器人位于一个 m x n 网格的左上角。机器人每次只能向下或向右移动一步。机器人试图达到网格的右下角。

问总共有多少条不同的路径？

示例:
- 输入: m = 3, n = 7
  输出: 28
  
- 输入: m = 3, n = 2
  输出: 3
  解释: 从左上角开始，总共有 3 条路径可以到达右下角。1. 向右 -> 向右 -> 向下 -> 向下；2. 向右 -> 向下 -> 向右 -> 向下；3. 向下 -> 向右 -> 向右 -> 向下
  
- 输入: m = 1, n = 1
  输出: 1

## 思路
使用**二维动态规划**：
1. dp[i][j] 表示到达位置 (i, j) 的路径数
2. 只能从上面或左边到达：dp[i][j] = dp[i-1][j] + dp[i][j-1]
3. 边界条件：第一行和第一列的所有位置只有 1 条路径
4. 空间优化：可以用一维数组替代二维数组
5. 时间复杂度：O(m*n)，空间复杂度：O(m*n) 或 O(n)

关键：递推关系简单，利用只能向下或向右的限制

## Python 代码

```python
def uniquePaths(m, n):
    # 1. 初始化二维 dp 数组
    dp = [[0] * n for _ in range(m)]
    
    # 2. 初始化第一行和第一列都为 1
    for i in range(m):
        dp[i][0] = 1
    for j in range(n):
        dp[0][j] = 1
    
    # 3. 填充 dp 数组
    for i in range(1, m):
        for j in range(1, n):
            # 4. 从上面或左边到达
            dp[i][j] = dp[i - 1][j] + dp[i][j - 1]
    
    return dp[m - 1][n - 1]

```

**空间优化版本**（一维 DP）：
```python
def uniquePaths_optimize(m, n):
    # 1. 初始化一维 dp 数组（表示一行）
    dp = [1] * n
    
    # 2. 从第二行开始遍历
    for i in range(1, m):
        # 3. 对每一列进行更新
        for j in range(1, n):
            # 4. 当前位置 = 上方（原值）+ 左方（已更新）
            dp[j] = dp[j] + dp[j - 1]
    
    return dp[n - 1]
```

## 易错点
1. **递推关系**：dp[i][j] = dp[i-1][j] + dp[i][j-1]
2. **边界初始化**：第一行和第一列的所有值都是 1
3. **循环范围**：从 (1, 1) 开始遍历
4. **空间优化**：一维数组可以按行覆盖，前一行的信息通过不覆盖第一列来保留
5. **返回值**：dp[m-1][n-1] 或 dp[n-1]（优化版）
