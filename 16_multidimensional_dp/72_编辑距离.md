# 72. 编辑距离

## 题目
给你两个单词 word1 和 word2，请返回将 word1 转换成 word2 所使用的最少操作数。

你可以对一个单词进行如下三种操作：
- 插入一个字符
- 删除一个字符
- 替换一个字符

示例:
- 输入: word1 = "horse", word2 = "ros"
  输出: 3
  解释: horse → rorse （将 'h' 替换为 'r'）→ rose （删除 'o'）→ ros （删除 'e'）
  
- 输入: word1 = "intention", word2 = "execution"
  输出: 5

## 思路
使用**二维动态规划**（莱文斯坦距离）：
1. dp[i][j] 表示将 word1 的前 i 个字符转换为 word2 的前 j 个字符的最少操作数
2. 递推关系：
   - 如果 word1[i-1] == word2[j-1]，则 dp[i][j] = dp[i-1][j-1]（不需要操作）
   - 否则 dp[i][j] = 1 + min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1])
     - dp[i-1][j] + 1：删除 word1[i-1]
     - dp[i][j-1] + 1：插入 word2[j-1]
     - dp[i-1][j-1] + 1：替换 word1[i-1] 为 word2[j-1]
3. 初始化：dp[0][j] = j（插入 j 个字符），dp[i][0] = i（删除 i 个字符）
4. 答案为 dp[m][n]

关键：三种操作对应三个来源状态，选择操作数最少的

## Python 代码

```python
def minDistance(word1, word2):
    # 1. 初始化 m, n
    m, n = len(word1), len(word2)
    
    # 2. 初始化 dp 数组
    # dp[i][j] = word1 前 i 个字符转换为 word2 前 j 个字符的最少操作数
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    # 3. 初始化第一行和第一列
    for i in range(m + 1):
        dp[i][0] = i  # 删除 word1 的前 i 个字符
    for j in range(n + 1):
        dp[0][j] = j  # 插入 word2 的前 j 个字符
    
    # 4. 填充 dp 数组
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            # 5. 比较 word1 的第 i-1 个字符和 word2 的第 j-1 个字符
            if word1[i - 1] == word2[j - 1]:
                # 6. 相同，不需要操作
                dp[i][j] = dp[i - 1][j - 1]
            else:
                # 7. 不同，选择三种操作中操作数最少的
                dp[i][j] = 1 + min(
                    dp[i - 1][j],      # 删除
                    dp[i][j - 1],      # 插入
                    dp[i - 1][j - 1]   # 替换
                )
    
    return dp[m][n]

```

**空间优化版本**（一维 DP）：
```python
def minDistance_optimize(word1, word2):
    # 1. 确保 word2 是较短的
    if len(word1) < len(word2):
        word1, word2 = word2, word1
    
    m, n = len(word1), len(word2)
    
    # 2. 初始化一维 dp 数组
    dp = list(range(n + 1))
    
    # 3. 遍历 word1 的每个字符
    for i in range(1, m + 1):
        prev = i
        for j in range(1, n + 1):
            # 4. 保存对角线的值
            temp = dp[j]
            
            if word1[i - 1] == word2[j - 1]:
                dp[j] = prev
            else:
                dp[j] = 1 + min(prev, dp[j], dp[j - 1])
            
            prev = temp
    
    return dp[n]
```

## 易错点
1. **dp 的定义**：dp[i][j] 对应 word1 的前 i 个字符（不是索引 i）
2. **索引转换**：word1[i-1] 对应 dp[i] 中的字符
3. **递推关系**：三种操作都需要考虑，选择最小值
4. **初始化**：dp[0][j] = j，dp[i][0] = i
5. **空间优化**：需要保存对角线的值（prev），因为同时需要三个来源值
