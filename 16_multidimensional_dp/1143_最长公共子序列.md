# 1143. 最长公共子序列

## 题目
给定两个字符串 text1 和 text2，返回这两个字符串的最长 公共子序列 的长度。如果不存在 公共子序列 ，返回 0 。

一个字符串的 子序列 是指这样一个新的字符串：它是由原字符串在不改变字符的相对顺序的情况下删除某些字符（也可以不删除任何字符）后组成的新字符串。

例如，"ace" 是 "abcde" 的子序列，但 "aec" 不是。

示例:
- 输入: text1 = "abcde", text2 = "ace"
  输出: 3
  解释: 最长公共子序列是 "ace"，长度为 3
  
- 输入: text1 = "abc", text2 = "abc"
  输出: 3
  
- 输入: text1 = "abc", text2 = "def"
  输出: 0

## 思路
使用**二维动态规划**：
1. dp[i][j] 表示 text1 的前 i 个字符与 text2 的前 j 个字符的最长公共子序列长度
2. 递推关系：
   - 如果 text1[i-1] == text2[j-1]，则 dp[i][j] = dp[i-1][j-1] + 1（该字符可以加入 LCS）
   - 否则 dp[i][j] = max(dp[i-1][j], dp[i][j-1])（选择不包含当前字符的更长子序列）
3. 初始化：dp[0][j] = 0，dp[i][0] = 0（空字符串与任何字符串的 LCS 为 0）
4. 答案为 dp[m][n]
5. 可以回溯找出具体的子序列

关键：两个字符串的比较问题，利用二维 DP 的对齐结构

## Python 代码

```python
def longestCommonSubsequence(text1, text2):
    # 1. 初始化 m, n
    m, n = len(text1), len(text2)
    
    # 2. 初始化 dp 数组
    # dp[i][j] = text1 前 i 个字符与 text2 前 j 个字符的最长公共子序列长度
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    # 3. 填充 dp 数组
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            # 4. 比较 text1 的第 i-1 个字符和 text2 的第 j-1 个字符
            if text1[i - 1] == text2[j - 1]:
                # 5. 相同，将该字符加入 LCS
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                # 6. 不同，选择更长的子序列
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    
    return dp[m][n]

```

**回溯找出具体的 LCS**：
```python
def longestCommonSubsequence_with_path(text1, text2):
    # 1. 计算 dp 数组（同上）
    m, n = len(text1), len(text2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if text1[i - 1] == text2[j - 1]:
                dp[i][j] = dp[i - 1][j - 1] + 1
            else:
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
    
    # 2. 回溯找出 LCS
    lcs = []
    i, j = m, n
    
    while i > 0 and j > 0:
        if text1[i - 1] == text2[j - 1]:
            # 3. 字符匹配，添加到 LCS
            lcs.append(text1[i - 1])
            i -= 1
            j -= 1
        elif dp[i - 1][j] > dp[i][j - 1]:
            # 4. 从上面来
            i -= 1
        else:
            # 5. 从左边来
            j -= 1
    
    # 6. 反转列表得到正确顺序
    return ''.join(reversed(lcs))
```

## 易错点
1. **dp 的定义**：dp[i][j] 对应 text1 的前 i 个字符，不是索引 i
2. **递推关系**：相同时 +1，不同时取最大值
3. **初始化**：第一行和第一列都初始化为 0
4. **索引转换**：text1[i-1] 对应 dp[i] 中的字符
5. **回溯方向**：沿着 dp 值较大的方向回溯，相等时选择左上角
