# 207. 课程表

## 题目
你这个学期有 numCourses 门课要上，标记为 0 到 numCourses - 1。

给你一个数组 prerequisites，其中 prerequisites[i] = [ai, bi]，表示如果要学习课程 ai，必须先完成课程 bi。

请你判断是否可能完成所有课程的学习？如果可以，返回 true；否则返回 false。

示例 1:
输入: numCourses = 2, prerequisites = [[1,0]]
输出: true
解释: 总共有 2 门课程。学习课程 1 之前，你需要完成课程 0。这是可能的。

示例 2:
输入: numCourses = 2, prerequisites = [[1,0],[0,1]]
输出: false
解释: 总共有 2 门课程。学习课程 1 之前，你需要完成课程 0；并且学习课程 0 之前，你还应先完成课程 1。这是不可能的。

## 思路
该题是**环检测问题**，使用**拓扑排序（DFS）**或**入度表**判断是否存在循环：

### 方案：DFS + 访问状态
1. 构建有向图，prerequisites[i] = [a, b] 表示从 b 指向 a 的边（必须先做 b 才能做 a）
2. 使用三种状态标记每个课程：
   - 0：未访问
   - 1：正在访问（当前 DFS 栈中）
   - 2：已完成
3. DFS 访问每个课程：
   - 如果遇到状态为 1 的课程，说明存在环，返回 False
   - 如果遇到状态为 2 的课程，跳过（已验证）
   - 对所有先修课程进行 DFS
4. 所有课程都通过验证则返回 True

## Python 代码

```python
class Solution:
    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        # 1. 构建图：course -> list of courses that depend on it
        graph = [[] for _ in range(numCourses)]
        state = [0] * numCourses  # 0: unvisited, 1: visiting, 2: visited
        
        for course, prereq in prerequisites:
            graph[prereq].append(course)
        
        # 2. DFS 检测环
        def has_cycle(course):
            if state[course] == 1:
                # 正在访问中，发现环
                return True
            if state[course] == 2:
                # 已经验证，无环
                return False
            
            # 3. 标记为正在访问
            state[course] = 1
            
            # 4. 检查所有依赖的课程
            for dependent in graph[course]:
                if has_cycle(dependent):
                    return True
            
            # 5. 标记为已完成
            state[course] = 2
            return False
        
        # 6. 检查所有课程
        for course in range(numCourses):
            if has_cycle(course):
                return False
        
        return True

```

## 易错点
1. **图的构造**：prerequisites[i] = [a, b] 表示先做 b 再做 a，所以是从 b 指向 a 的有向边
2. **三种状态的含义**：0 未访问、1 正在访问、2 已完成，这三种状态的区分很重要
3. **环的判断**：如果在 DFS 栈中遇到状态为 1 的节点，说明存在环
4. **完整遍历**：需要对所有课程进行 DFS，因为图可能不连通
