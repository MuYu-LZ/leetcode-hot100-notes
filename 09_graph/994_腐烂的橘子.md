# 994. 腐烂的橘子

## 题目
在给定的 m x n 网格 grid 中，每个单元格可以有以下三个值之一：
- 0 代表空单元格
- 1 代表新鲜橘子
- 2 代表腐烂的橘子

每分钟，腐烂的橘子**相邻的每一个新鲜橘子都会变成腐烂**。

返回 直到单元格中没有新鲜橘子为止所必须经过的最少分钟数。如果不可能，返回 -1。

示例 1:
输入: grid = [[2,1,1],[1,1,0],[0,1,1]]
输出: 4

示例 2:
输入: grid = [[2,1,1],[0,1,1],[1,0,1]]
输出: -1

## 思路
该题使用**广度优先搜索（BFS）**模拟腐烂过程：
1. 初始化：将所有腐烂的橘子（值为 2）放入队列，统计新鲜橘子（值为 1）的个数
2. 逐分钟扩展：
   - 从队列中取出本分钟内腐烂的所有橘子
   - 将其四个方向的新鲜橘子变为腐烂，并加入队列（下一分钟处理）
   - 减少新鲜橘子计数
3. 返回结果：
   - 如果最后还有新鲜橘子，返回 -1
   - 否则返回经过的分钟数

## Python 代码

```python
from collections import deque

class Solution:
    def orangesRotting(self, grid: List[List[int]]) -> int:
        rows, cols = len(grid), len(grid[0])
        queue = deque()
        fresh_count = 0
        
        # 1. 初始化：将所有腐烂的橘子加入队列
        for i in range(rows):
            for j in range(cols):
                if grid[i][j] == 2:
                    queue.append((i, j))
                elif grid[i][j] == 1:
                    fresh_count += 1
        
        # 2. 边界情况：没有新鲜橘子
        if fresh_count == 0:
            return 0
        
        # 3. 方向数组：上下左右
        directions = [(0, 1), (0, -1), (1, 0), (-1, 0)]
        minutes = 0
        
        # 4. BFS 模拟腐烂过程
        while queue and fresh_count > 0:
            # 5. 每一分钟处理队列中的所有橘子
            minutes += 1
            queue_size = len(queue)
            
            for _ in range(queue_size):
                r, c = queue.popleft()
                
                # 6. 检查四个方向
                for dr, dc in directions:
                    nr, nc = r + dr, c + dc
                    
                    # 7. 检查边界和新鲜橘子
                    if 0 <= nr < rows and 0 <= nc < cols and grid[nr][nc] == 1:
                        # 8. 新鲜橘子变为腐烂
                        grid[nr][nc] = 2
                        queue.append((nr, nc))
                        fresh_count -= 1
        
        # 9. 返回结果
        return minutes if fresh_count == 0 else -1

```

## 易错点
1. **初始化的完整性**：需要同时找出所有腐烂的橘子和新鲜橘子的个数
2. **分钟的计数**：只有当处理了新鲜橘子后才能增加分钟数
3. **每分钟处理一层**：通过记录当前队列大小，确保按分钟为单位处理
4. **边界检查**：检查坐标是否有效且目标是新鲜橘子
5. **最后的判断**：最后仍有新鲜橘子说明无法全部腐烂，返回 -1
