# 094. 二叉树的中序遍历

## 题目
给定一个二叉树的根节点 root ，返回它的中序遍历。

## 思路
使用**递归或迭代（栈）**实现中序遍历（左-根-右）。

## Python 代码

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def inorderTraversal(self, root: Optional[TreeNode]) -> List[int]:
        # 递归方法
        result = []
        def dfs(node):
            if not node:
                return
            dfs(node.left)
            result.append(node.val)
            dfs(node.right)
        dfs(root)
        return result

```

## 易错点
1. **遍历顺序**：左-根-右
2. **递归基础情况**：节点为 None 时返回
