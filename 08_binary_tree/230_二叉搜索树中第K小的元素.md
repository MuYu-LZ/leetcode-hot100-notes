# 230. 二叉搜索树中第K小的元素

## 思路
中序遍历得到升序序列，第 K 个即答案。

## Python 代码

```python
class Solution:
    def kthSmallest(self, root: Optional[TreeNode], k: int) -> int:
        self.count = 0
        self.result = 0
        def dfs(node):
            if not node:
                return
            dfs(node.left)
            self.count += 1
            if self.count == k:
                self.result = node.val
                return
            dfs(node.right)
        dfs(root)
        return self.result
```
