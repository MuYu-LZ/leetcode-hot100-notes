# 437. 路径总和 III

## 思路
DFS + 前缀和：每个节点出发，找路径和为 target 的路径数。

## Python 代码

```python
class Solution:
    def pathSum(self, root: Optional[TreeNode], targetSum: int) -> int:
        self.count = 0
        def dfs(node, current_sum):
            if not node:
                return
            current_sum += node.val
            if current_sum == targetSum:
                self.count += 1
            dfs(node.left, current_sum)
            dfs(node.right, current_sum)
        def traverse(node):
            if not node:
                return
            dfs(node, 0)
            traverse(node.left)
            traverse(node.right)
        traverse(root)
        return self.count
```
