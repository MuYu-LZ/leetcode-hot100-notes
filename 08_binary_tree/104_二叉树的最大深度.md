# 104. 二叉树的最大深度

## 题目
给定一个二叉树，找出其最大深度。最大深度是从根节点到最远叶子节点的最长路径上的节点数。

## 思路
递归：对于每个节点，最大深度 = 1 + max(左子树深度, 右子树深度)

## Python 代码

```python
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if not root:
            return 0
        return 1 + max(self.maxDepth(root.left), self.maxDepth(root.right))
```

## 易错点
1. 递归基础情况：None 返回 0
2. 当前深度 = 1 + 两子树较大深度
