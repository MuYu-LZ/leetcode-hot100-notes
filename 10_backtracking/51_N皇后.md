# 051. N 皇后

## 题目
按照国际象棋的规则，皇后可以攻击与之处于同一行、同一列或同一斜线上的棋子。

n 皇后问题 研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后之间不能相互攻击。

给你一个整数 n ，返回所有不同的 n 皇后问题的解决方案。

每一种解法包含一个明确的 n 皇后问题的棋盘配置，其中 'Q' 表示一个皇后，'.' 表示一个空位。

## 思路
该题使用**回溯**逐行放置皇后，并检查冲突：
1. 使用三个集合记录已占用的列、主对角线和副对角线
2. 逐行放置皇后，对于每一行，尝试在不同的列放置
3. 检查该位置是否与已放置的皇后冲突（使用集合快速判断）
4. 如果放置成功，递归处理下一行；否则回溯尝试其他列

## Python 代码

```python
class Solution:
    def solveNQueens(self, n: int) -> List[List[str]]:
        result = []
        board = [['.' for _ in range(n)] for _ in range(n)]
        cols = set()          # 已占用的列
        diag1 = set()         # 已占用的主对角线（左上到右下）
        diag2 = set()         # 已占用的副对角线（右上到左下）
        
        def is_safe(row, col):
            # 检查列和两条对角线
            return col not in cols and (row - col) not in diag1 and (row + col) not in diag2
        
        def backtrack(row):
            # 1. 基础情况：所有行都放置了皇后
            if row == n:
                result.append([''.join(r) for r in board])
                return
            
            # 2. 尝试在当前行的每一列放置皇后
            for col in range(n):
                if is_safe(row, col):
                    # 3. 放置皇后
                    board[row][col] = 'Q'
                    cols.add(col)
                    diag1.add(row - col)
                    diag2.add(row + col)
                    
                    # 4. 递归处理下一行
                    backtrack(row + 1)
                    
                    # 5. 回溯
                    board[row][col] = '.'
                    cols.remove(col)
                    diag1.remove(row - col)
                    diag2.remove(row + col)
        
        backtrack(0)
        return result

```

## 易错点
1. **对角线的表示**：
   - 主对角线：row - col 相同
   - 副对角线：row + col 相同
2. **集合的使用**：用集合快速判断是否冲突，比列表更高效
3. **字符串转换**：最后输出时需要将列表转换为字符串（[''.join(r) for r in board]）
4. **回溯的完整性**：放置后的所有修改都必须回溯（列、两条对角线、棋盘）
5. **时间复杂度**：O(N!)，由于约束条件的剪枝
