# 039. 组合总和

## 题目
给定一个无重复元素的正整数数组 candidates 和一个正整数 target，找出 candidates 中所有可以使数字和为目标数 target 的唯一组合。

candidates 中的数字可以无限制重复被选取。如果至少一个数字的被选数量不同，则两种组合是不同的。

示例 1:
输入: candidates = [2,3,6,7], target = 7
输出: [[2,2,3],[7]]

示例 2:
输入: candidates = [2,3,5], target = 5
输出: [[2,3],[5]]

## 思路
该题使用**回溯**寻找所有组合，关键特点是**数字可以重复使用**：
1. 对于每个候选数字，可以选择使用或不使用
2. 如果选择使用，可以继续使用同一个数字（通过保持 start 索引不变）
3. 剪枝优化：如果当前和已经超过目标，停止继续
4. 当和等于目标时，记录该组合

## Python 代码

```python
class Solution:
    def combinationSum(self, candidates: List[int], target: int) -> List[List[int]]:
        result = []
        
        def backtrack(start, current, remaining):
            # 1. 基础情况：和达到目标
            if remaining == 0:
                result.append(current[:])
                return
            
            # 2. 剪枝：和已超过目标
            if remaining < 0:
                return
            
            # 3. 尝试从 start 开始的每个候选数字
            for i in range(start, len(candidates)):
                # 4. 选择当前数字
                current.append(candidates[i])
                
                # 5. 递归：可以重复使用当前数字（start 不变）
                backtrack(i, current, remaining - candidates[i])
                
                # 6. 回溯
                current.pop()
        
        backtrack(0, [], target)
        return result

```

## 易错点
1. **重复使用**：递归时 backtrack(i, ...) 而不是 backtrack(i + 1, ...)，允许重复使用同一个数字
2. **start 索引**：start 参数避免了有重复的组合（例如 [2, 3] 和 [3, 2] 只会保留一种）
3. **剪枝条件**：remaining < 0 时可以提前返回，提高效率
4. **副本添加**：result.append(current[:]) 添加副本而非引用
5. **时间复杂度**：O(N^(T/M))，其中 T 是目标值，M 是最小候选数
