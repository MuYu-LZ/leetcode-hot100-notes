# 017. 电话号码的字母组合

## 题目
给定一个仅包含数字 2-9 的字符串，返回所有它能表示的字母组合。回答可以按任意顺序返回。

给出数字到字母的映射如下（就像在电话按键上一样）：
2:'abc', 3:'def', 4:'ghi', 5:'jkl', 6:'mno', 7:'pqrs', 8:'tuv', 9:'wxyz'

示例 1:
输入: digits = "23"
输出: ["ad","ae","af","bd","be","bf","cd","ce","cf"]

示例 2:
输入: digits = ""
输出: []

示例 3:
输入: digits = "2"
输出: ["a","b","c"]

## 思路
该题使用**回溯**生成所有字母组合：
1. 创建数字到字母的映射表
2. 对于每个数字，依次选择对应的字母
3. 递归处理下一个数字
4. 当处理完所有数字时，得到一个完整的字母组合

## Python 代码

```python
class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        # 1. 边界情况
        if not digits:
            return []
        
        # 2. 数字到字母的映射
        mapping = {
            '2': 'abc', '3': 'def', '4': 'ghi', '5': 'jkl',
            '6': 'mno', '7': 'pqrs', '8': 'tuv', '9': 'wxyz'
        }
        
        result = []
        
        def backtrack(index, current):
            # 3. 基础情况：处理完所有数字
            if index == len(digits):
                result.append(current)
                return
            
            # 4. 获取当前数字对应的字母
            letters = mapping[digits[index]]
            
            # 5. 尝试每个字母
            for letter in letters:
                # 6. 选择
                # 7. 递归处理下一个数字
                backtrack(index + 1, current + letter)
                # 8. 回溯（由于字符串不可变，自动进行）
        
        backtrack(0, '')
        return result

```

## 易错点
1. **空字符串处理**：digits 为空时应返回空列表
2. **映射表**：必须包含所有数字 2-9 的对应字母
3. **索引管理**：index 表示当前处理的数字位置
4. **字符串拼接**：current + letter 创建新字符串，无需显式回溯
5. **时间复杂度**：O(4^n)，最坏情况下（全是 7 或 9）有 4^n 种组合
