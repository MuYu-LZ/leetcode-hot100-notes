# 131. 分割回文串

## 题目
给你一个字符串 s，请你将 s 分割成一些子串，使每个子串都是回文串。返回 s 所有可能的分割方案。

示例 1:
输入: s = "aab"
输出: [["a","a","b"],["aa","b"]]

示例 2:
输入: s = "nitin"
输出: [["n","i","t","i","n"]]

## 思路
该题使用**回溯**分割字符串，并检查每个子串是否是回文：
1. 对于每个起始位置，尝试所有可能的分割点
2. 检查分割出的子串是否是回文
3. 如果是，递归处理剩余部分
4. 如果分割到了字符串末尾，记录一个完整的分割方案

## Python 代码

```python
class Solution:
    def partition(self, s: str) -> List[List[str]]:
        result = []
        
        def is_palindrome(sub):
            # 判断字符串是否是回文
            return sub == sub[::-1]
        
        def backtrack(start, current):
            # 1. 基础情况：已分割到字符串末尾
            if start == len(s):
                result.append(current[:])
                return
            
            # 2. 尝试每个分割点
            for end in range(start + 1, len(s) + 1):
                # 3. 获取当前分割的子串
                substring = s[start:end]
                
                # 4. 检查是否是回文
                if is_palindrome(substring):
                    # 5. 选择：添加到当前分割
                    current.append(substring)
                    
                    # 6. 递归处理剩余部分
                    backtrack(end, current)
                    
                    # 7. 回溯
                    current.pop()
        
        backtrack(0, [])
        return result

```

## 易错点
1. **回文检查**：sub == sub[::-1] 是最简单的方式，也可以用双指针优化
2. **分割范围**：range(start + 1, len(s) + 1) 确保每次至少分割一个字符
3. **副本添加**：result.append(current[:]) 添加当前分割方案的副本
4. **时间复杂度**：O(N * 2^N)，回文字符串数量指数级增长
5. **优化**：可以用动态规划预处理回文判断，避免重复计算
