# 079. 单词搜索

## 题目
给定一个 m x n 二维字符网格 board 和一个字符串单词 word 。如果 word 存在于网格中，返回 true ；否则，返回 false 。

单词必须按照字母顺序，通过相邻的单元格内的字母构成，其中"相邻"单元格是那些水平相邻或竖直相邻的单元格。同一个单元格内的字母不允许被重复使用。

## 思路
该题使用**回溯 + DFS**搜索单词：
1. 遍历网格，找到与单词首字母匹配的单元格
2. 从该位置开始 DFS，逐个匹配单词的后续字母
3. 在 DFS 中，标记当前位置已访问，然后递归检查四个方向
4. 如果成功匹配整个单词，返回 True；否则回溯

## Python 代码

```python
class Solution:
    def exist(self, board: List[List[str]], word: str) -> bool:
        rows, cols = len(board), len(board[0])
        
        def dfs(r, c, index):
            # 1. 基础情况：已匹配整个单词
            if index == len(word):
                return True
            
            # 2. 边界检查和字符检查
            if r < 0 or r >= rows or c < 0 or c >= cols or board[r][c] != word[index]:
                return False
            
            # 3. 标记当前位置已访问（使用临时变量）
            board[r][c] = '#'
            
            # 4. 递归检查四个方向
            result = (dfs(r + 1, c, index + 1) or
                      dfs(r - 1, c, index + 1) or
                      dfs(r, c + 1, index + 1) or
                      dfs(r, c - 1, index + 1))
            
            # 5. 回溯：恢复原字符
            board[r][c] = word[index]
            
            return result
        
        # 6. 遍历网格，寻找单词的起点
        for i in range(rows):
            for j in range(cols):
                if board[i][j] == word[0]:
                    if dfs(i, j, 0):
                        return True
        
        return False

```

## 易错点
1. **访问标记**：使用 board[r][c] = '#' 标记访问，然后在回溯时恢复，避免额外的访问集合
2. **边界和字符检查**：在同一个条件中进行，简化逻辑
3. **index 管理**：index 表示当前匹配的单词位置
4. **短路求值**：使用 or 连接四个方向的 dfs，一旦找到就返回 True
5. **时间复杂度**：O(m * n * 4^l)，其中 l 是单词长度
