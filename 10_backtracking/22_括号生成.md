# 022. 括号生成

## 题目
数字 n 代表生成括号的对数，请你设计一个函数，用于能生成所有可能的并且有效的括号组合。

示例 1:
输入: n = 3
输出: ["((()))","(()())","(())()","()(())","()()()"]

示例 2:
输入: n = 1
输出: ["()"]

## 思路
该题使用**回溯**生成所有有效括号组合：
1. 维护两个计数器：open（已放置的左括号数）和 close（已放置的右括号数）
2. 在每个位置，如果可以放置左括号或右括号，进行回溯尝试
3. 可以放置条件：
   - 左括号：已放置数 < n
   - 右括号：已放置数 < open（右括号不能超过左括号）

## Python 代码

```python
class Solution:
    def generateParenthesis(self, n: int) -> List[str]:
        result = []
        
        def backtrack(current, open_count, close_count):
            # 1. 基础情况：生成了 n 对括号
            if len(current) == 2 * n:
                result.append(current)
                return
            
            # 2. 尝试添加左括号
            if open_count < n:
                backtrack(current + '(', open_count + 1, close_count)
            
            # 3. 尝试添加右括号
            if close_count < open_count:
                backtrack(current + ')', open_count, close_count + 1)
        
        backtrack('', 0, 0)
        return result

```

## 易错点
1. **条件判断的准确性**：
   - 左括号：open_count < n
   - 右括号：close_count < open_count（不能超过已放置的左括号）
2. **基础情况**：len(current) == 2 * n（因为共有 n 对括号，2n 个字符）
3. **有效性保证**：通过在回溯中限制条件，确保生成的都是有效括号组合
4. **时间复杂度**：O(4^n / n^(3/2))，Catalan 数列
