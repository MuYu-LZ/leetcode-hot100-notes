# 078. 子集

## 题目
给你一个整数数组 nums，其中没有重复元素。返回该数组的所有子集（幂集）。

解集不能包含重复的子集。你可以按任意顺序返回解集。

示例 1:
输入: nums = [1,2,3]
输出: [[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]

示例 2:
输入: nums = [1]
输出: [[],[1]]

## 思路
该题使用**回溯**生成所有子集：
1. 对于每个元素，有两种选择：**包含或不包含**
2. 递归构建子集：
   - 从当前位置开始，尝试添加后续元素
   - 每次添加一个元素后，递归处理剩余元素
   - 回溯移除最后添加的元素，继续尝试其他选项
3. 每一步都将当前子集添加到结果中

## Python 代码

```python
class Solution:
    def subsets(self, nums: List[int]) -> List[List[int]]:
        result = []
        
        def backtrack(start, current):
            # 1. 每个状态都是一个有效的子集
            result.append(current[:])
            
            # 2. 尝试添加后续元素
            for i in range(start, len(nums)):
                # 3. 选择
                current.append(nums[i])
                
                # 4. 递归，从下一个元素开始
                backtrack(i + 1, current)
                
                # 5. 回溯
                current.pop()
        
        backtrack(0, [])
        return result

```

## 易错点
1. **每个状态都是子集**：不仅是到叶子节点，中间过程中的每个状态都是一个有效子集
2. **start 索引的重要性**：避免添加重复的子集，start 确保只添加后续元素
3. **副本的添加**：result.append(current[:]) 添加当前列表的副本
4. **时间复杂度**：O(n * 2^n)，有 2^n 个子集，每个构造需要 O(n)
