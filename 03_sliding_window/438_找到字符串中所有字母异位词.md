# 438. 找到字符串中所有字母异位词

## 题目
给定两个字符串 s 和 p ，找到 s 中所有 p 的异位词的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

异位词指由相同字母重新排列形成的字符串。

示例 1:
输入: s = "cbaebabacd", p = "abc"
输出: [0,6]
解释: 
起始索引等于 0 的子串是 "cba"，它是 "abc" 的异位词。
起始索引等于 6 的子串是 "bac"，它是 "abc" 的异位词。

示例 2:
输入: s = "abab", p = "ab"
输出: [0,1,2]
解释:
起始索引等于 0 的子串是 "ab"，它是 "ab" 的异位词。
起始索引等于 1 的子串是 "ba"，它是 "ab" 的异位词。
起始索引等于 2 的子串是 "ab"，它是 "ab" 的异位词。

## 思路
该题使用**滑动窗口**结合**字符频率**对比：
1. 创建两个频率表：一个记录目标字符串 p 的字符频率，另一个记录窗口内的字符频率
2. 使用双指针维护一个大小固定为 len(p) 的滑动窗口
3. 窗口右端逐个加入新字符，窗口左端在窗口大小超过 len(p) 时移出旧字符
4. 每当窗口大小等于 len(p) 时，比较两个频率表是否相同，若相同则该起始位置是异位词

关键思想：通过固定大小的滑动窗口和频率表对比，可以一次遍历找到所有异位词。

## Python 代码

```python
class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        # 1. 如果 s 长度小于 p，不可能有异位词
        if len(s) < len(p):
            return []
        
        # 2. 创建频率表
        p_count = {}
        window_count = {}
        for char in p:
            p_count[char] = p_count.get(char, 0) + 1
        
        result = []
        
        # 3. 滑动窗口遍历
        for i in range(len(s)):
            # 4. 将右端字符加入窗口
            char = s[i]
            window_count[char] = window_count.get(char, 0) + 1
            
            # 5. 当窗口大小超过 p 的长度时，移除左端字符
            if i >= len(p):
                left_char = s[i - len(p)]
                if window_count[left_char] == 1:
                    del window_count[left_char]
                else:
                    window_count[left_char] -= 1
            
            # 6. 当窗口大小等于 p 的长度时，比较频率表
            if window_count == p_count:
                result.append(i - len(p) + 1)
        
        return result

```

## 易错点
1. **窗口大小的控制**：窗口大小必须始终等于 len(p)，不能更大
2. **频率表的对比**：直接比较两个字典是否相等，而不是逐个字符比较
3. **边界处理**：需要在 i >= len(p) 时才开始移除左端字符
4. **删除字符的处理**：当某字符频率变为 0 时，从字典中删除它（而不是保留为 0），这样字典比较才会相等
