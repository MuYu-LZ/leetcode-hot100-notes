# 003. 无重复字符的最长子串

## 题目
给定一个字符串 s ，请你找出其中不含有重复字符的最长子串的长度。

示例 1：
输入: s = "abcabcbb"
输出: 3
解释: 最长子串是 "abc"，长度为 3。

示例 2：
输入: s = "bbbbb"
输出: 1
解释: 最长子串是 "b"，长度为 1。

示例 3：
输入: s = "pwwkew"
输出: 3
解释: 最长子串是 "wke"，长度为 3。

## 思路
该题使用**滑动窗口**策略：
1. 使用双指针（left 和 right）维护一个窗口，窗口内没有重复字符
2. 使用哈希表（字典）记录窗口内各字符及其最后出现的位置
3. 右指针逐个遍历字符：
   - 如果字符不在哈希表中或在窗口外，直接加入窗口
   - 如果字符已在窗口内，移动左指针，使其越过重复字符的上一个位置
4. 记录整个过程中最大的窗口长度

关键思想：窗口的左边界需要更新为重复字符上一次出现位置之后。

## Python 代码

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        # 1. 初始化哈希表存储字符和对应的最后位置
        char_index = {}
        max_len = 0
        left = 0
        
        # 2. 右指针遍历字符串
        for right in range(len(s)):
            # 3. 如果字符已在窗口内出现过
            if s[right] in char_index and char_index[s[right]] >= left:
                # 4. 移动左指针到重复字符的下一个位置
                left = char_index[s[right]] + 1
            
            # 5. 更新当前字符的最后位置
            char_index[s[right]] = right
            
            # 6. 更新最长子串长度
            max_len = max(max_len, right - left + 1)
        
        return max_len

```

## 易错点
1. **哈希表的作用**：存储的是字符最后出现的位置，而不仅仅是字符本身
2. **左指针的更新**：需要比较 char_index[s[right]] 和 left，确保该字符在当前窗口内才更新左指针
3. **位置更新的时机**：要先更新左指针，再更新哈希表中的位置
4. **窗口长度计算**：right - left + 1 是包含两个端点的长度
