# 189. 轮转数组

## 题目
给定一个整数数组 nums，将数组中的元素向右轮转 k 个位置，其中 k 是非负数。

示例 1:
输入: nums = [1,2,3,4,5,6,7], k = 3
输出: [5,6,7,1,2,3,4]
解释:
向右轮转 1 步: [7,1,2,3,4,5,6]
向右轮转 2 步: [6,7,1,2,3,4,5]
向右轮转 3 步: [5,6,7,1,2,3,4]

示例 2:
输入: nums = [-1,-100,3,99], k = 2
输出: [3,99,-1,-100]
解释: 
向右轮转 1 步: [99,-1,-100,3]
向右轮转 2 步: [3,99,-1,-100]

## 思路
该题使用**数组反转**的巧妙方法，时间复杂度为 O(n)，空间复杂度为 O(1)：
1. 先将 k 对数组长度取模，避免重复轮转
2. 反转整个数组
3. 反转前 k 个元素
4. 反转剩余的 n - k 个元素

这样就实现了向右轮转 k 个位置。原理：
- 原数组：[1, 2, 3, 4, 5, 6, 7]（k=3）
- 第1步反转全数组：[7, 6, 5, 4, 3, 2, 1]
- 第2步反转前3个：[5, 6, 7, 4, 3, 2, 1]
- 第3步反转后4个：[5, 6, 7, 1, 2, 3, 4] ✓

## Python 代码

```python
class Solution:
    def rotate(self, nums: List[int], k: int) -> None:
        # 1. 辅助函数：反转数组的某个区间
        def reverse(start, end):
            while start < end:
                nums[start], nums[end] = nums[end], nums[start]
                start += 1
                end -= 1
        
        # 2. 处理 k 大于数组长度的情况
        n = len(nums)
        k = k % n
        
        # 3. 三次反转实现轮转
        reverse(0, n - 1)           # 反转整个数组
        reverse(0, k - 1)           # 反转前 k 个元素
        reverse(k, n - 1)           # 反转剩余元素
```

## 易错点
1. **k 的模运算**：k 可能大于数组长度，需要对 n 取模
2. **反转函数的边界**：注意反转区间的起点和终点，确保不越界
3. **三次反转的顺序**：顺序固定，不能改变
4. **空间复杂度**：相比创建新数组，这种方法只使用常数级额外空间
