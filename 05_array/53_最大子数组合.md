# 053. 最大子数组合

## 题目
给你一个整数数组 nums ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

子数组是数组中的一个连续部分。

示例 1:
输入: nums = [-2,1,-3,4,-1,2,1,-5,4]
输出: 6
解释: 连续子数组 [4,-1,2,1] 有最大和 6。

示例 2:
输入: nums = [5]
输出: 5

示例 3:
输入: nums = [5,4,-1,7]
输出: 16

## 思路
该题使用**贪心算法**（Kadane's Algorithm）：
1. 维护两个变量：
   - current_sum：表示以当前位置结尾的最大子数组和
   - max_sum：全局最大子数组和
2. 对于每个元素，决定是否加入之前的子数组：
   - 如果加入前的子数组会使总和变小，就从当前元素重新开始
   - 否则，继续累加
3. 这相当于在每个位置问："是继续累加，还是重新开始？"

关键思想：current_sum = max(current_sum + nums[i], nums[i])，通过这个递推关系，一次遍历即可求解。

## Python 代码

```python
class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        # 1. 初始化当前和与全局最大值
        current_sum = nums[0]
        max_sum = nums[0]
        
        # 2. 从第二个元素开始遍历
        for i in range(1, len(nums)):
            # 3. 决定是加入前面的子数组，还是从当前元素重新开始
            current_sum = max(current_sum + nums[i], nums[i])
            # 4. 更新全局最大值
            max_sum = max(max_sum, current_sum)
        
        return max_sum

```

## 易错点
1. **current_sum 的含义**：表示以当前位置结尾的最大子数组和，而不是某个任意子数组的和
2. **max() 的使用**：max(current_sum + nums[i], nums[i]) 决定了是否要将当前元素添加到前面的子数组
3. **初始化**：必须初始化为第一个元素，而不是 0（否则会影响全负数的情况）
4. **一次遍历即可**：时间复杂度 O(n)，空间复杂度 O(1)，非常高效
