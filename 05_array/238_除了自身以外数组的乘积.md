# 238. 除了自身以外数组的乘积

## 题目
给你一个整数数组 nums，返回数组 answer，其中 answer[i] 等于 nums 中除 nums[i] 之外其余各元素的乘积。

题目数据保证数组之外的所有前缀乘积和后缀乘积都可以用 32 位整数表示。

请不要使用除法，且在 O(n) 时间复杂度内完成此题。

示例 1:
输入: nums = [1,2,3,4]
输出: [24,12,8,6]

示例 2:
输入: nums = [-1,1,0,-3,3]
输出: [0,0,9,0,0]

## 思路
该题使用**前缀乘积和后缀乘积**的思想：
1. 对于 answer[i]，其值应该是 nums[0...i-1] 的乘积 × nums[i+1...n-1] 的乘积
2. 第一遍遍历：计算前缀乘积（即 nums[0...i-1] 的乘积），存入 answer 数组
3. 第二遍遍历：从右到左计算后缀乘积（即 nums[i+1...n-1] 的乘积），与 answer[i] 相乘

这样避免了使用除法，也不需要额外的数组存储前缀和后缀乘积。

## Python 代码

```python
class Solution:
    def productExceptSelf(self, nums: List[int]) -> List[int]:
        n = len(nums)
        # 1. 初始化结果数组
        answer = [1] * n
        
        # 2. 第一遍：计算前缀乘积
        # answer[i] = nums[0] * nums[1] * ... * nums[i-1]
        for i in range(1, n):
            answer[i] = answer[i - 1] * nums[i - 1]
        
        # 3. 第二遍：从右到左计算后缀乘积并与前缀乘积相乘
        suffix = 1  # 后缀乘积
        for i in range(n - 1, -1, -1):
            # 4. answer[i] = 前缀乘积 * 后缀乘积
            answer[i] = answer[i] * suffix
            # 5. 更新后缀乘积（加入当前元素）
            suffix *= nums[i]
        
        return answer

```

## 易错点
1. **两遍遍历的含义**：第一遍是前缀乘积，第二遍是后缀乘积，两者相乘得到最终答案
2. **初始化**：answer 数组初始化为全 1，这样可以直接与乘积相乘
3. **后缀乘积的计算**：从右到左遍历，suffix 在使用后才更新
4. **不使用除法**：通过分离前缀和后缀乘积，巧妙地避免了除法（包括处理 0 的情况）
