# 041. 缺失的第一个正数

## 题目
给你一个未排序的整数数组 nums ，请你找出其中没有出现的最小正整数。

请你实现时间复杂度为 O(n)，空间复杂度为 O(1) 的解决方案。

示例 1:
输入：nums = [1,2,0]
输出：3

示例 2:
输入：nums = [3,4,-1,1]
输出：2

示例 3:
输入：nums = [7,8,9]
输出：1

## 思路
该题的关键在于充分利用数组本身作为哈希表：
1. 观察到对于长度为 n 的数组，最小缺失的正整数只能在 1 到 n 之间
   - 如果 1 到 n 都存在，答案是 n+1
   - 否则，答案一定在 1 到 n 之间
2. 将数组作为哈希表：对于值 x（1 ≤ x ≤ n），将其放在索引 x-1 的位置
3. 第一遍遍历：将每个数字放在合适的位置
4. 第二遍遍历：找到第一个位置 i，其值不是 i+1，答案就是 i+1

## Python 代码

```python
class Solution:
    def firstMissingPositive(self, nums: List[int]) -> int:
        n = len(nums)
        
        # 1. 第一遍遍历：将每个数放在合适的位置
        for i in range(n):
            # 2. 如果当前数 x 满足 1 <= x <= n，且 x 不在位置 x-1
            while 1 <= nums[i] <= n and nums[i] != nums[nums[i] - 1]:
                # 3. 交换 nums[i] 和 nums[nums[i] - 1]
                target_idx = nums[i] - 1
                nums[i], nums[target_idx] = nums[target_idx], nums[i]
        
        # 4. 第二遍遍历：找到第一个不在合适位置的数
        for i in range(n):
            if nums[i] != i + 1:
                return i + 1
        
        # 5. 如果所有位置都正确，答案是 n+1
        return n + 1

```

## 易错点
1. **值的范围限制**：只有 1 ≤ x ≤ n 的数才需要放在特定位置，其他数可以忽略
2. **while 循环的条件**：需要检查 nums[nums[i] - 1] 是否等于 nums[i]，避免无限循环
3. **交换的安全性**：交换前确保目标位置在有效范围内，且交换能让问题往解决方向靠近
4. **最后的答案**：如果所有位置都正确，说明 1 到 n 都存在，答案是 n+1
