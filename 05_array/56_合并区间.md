# 056. 合并区间

## 题目
以数组 intervals 表示若干个区间的集合，其中单个区间为 intervals[i] = [starti, endi] 。请你合并所有重叠的区间，并返回一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间。

示例 1:
输入: intervals = [[1,3],[2,6],[8,10],[15,18]]
输出: [[1,6],[8,10],[15,18]]
解释: 区间 [1,3] 和 [2,6] 重叠，合并为 [1,6]。

示例 2:
输入: intervals = [[1,4],[4,5]]
输出: [[1,5]]
解释: 区间 [1,4] 和 [4,5] 可被视为重叠区间。

## 思路
该题使用**排序 + 贪心**的策略：
1. 首先按区间的起点进行排序
2. 遍历排序后的区间：
   - 如果当前区间的起点大于之前合并区间的终点，说明不重叠，将其作为新的区间加入结果
   - 否则，更新之前合并区间的终点为两个区间终点的较大值
3. 通过一次遍历完成所有合并操作

关键思想：排序后，只需比较相邻区间是否重叠，简化了问题。

## Python 代码

```python
class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        # 1. 边界处理
        if not intervals:
            return []
        
        # 2. 按区间起点排序
        intervals.sort(key=lambda x: x[0])
        
        # 3. 初始化结果列表，添加第一个区间
        result = [intervals[0]]
        
        # 4. 遍历剩余的区间
        for i in range(1, len(intervals)):
            # 5. 获取结果中最后一个区间（已合并的区间）
            last_merged = result[-1]
            current = intervals[i]
            
            # 6. 检查是否重叠
            if current[0] <= last_merged[1]:
                # 7. 重叠：更新最后一个区间的终点
                last_merged[1] = max(last_merged[1], current[1])
            else:
                # 8. 不重叠：将当前区间添加到结果
                result.append(current)
        
        return result

```

## 易错点
1. **排序的关键**：按起点排序（key=lambda x: x[0]），这样可以线性地处理重叠问题
2. **重叠的判断**：current[0] <= last_merged[1]（当前区间的起点不超过上一个区间的终点，则重叠）
3. **终点的更新**：使用 max() 取两个终点的较大值，确保合并后的区间覆盖两个原区间
4. **原地修改vs新建区间**：代码直接修改了 result[-1]，这是可以的；也可以重新创建新区间加入结果
