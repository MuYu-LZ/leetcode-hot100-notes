# 169. 多数元素

## 题目
给定一个大小为 n 的数组，找到其中的多数元素。多数元素是指在数组中出现次数 **大于 ⌊ n/2 ⌋** 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。

示例:
- 输入: nums = [3,2,3]
  输出: 3
  
- 输入: nums = [2,2,1,1,1,2,2]
  输出: 2
  
- 输入: nums = [1]
  输出: 1

## 思路
**方法一：哈希表**（O(n) 时间，O(n) 空间）：
统计每个元素的出现次数，返回出现次数大于 n/2 的元素。

**方法二：排序**（O(n log n) 时间，O(1) 空间）：
排序后，中间位置的元素一定是多数元素。

**方法三：摩尔投票算法**（O(n) 时间，O(1) 空间）- 推荐：
1. 维护一个候选数和计数器
2. 当计数器为 0 时，选择当前数为新候选
3. 如果当前数等于候选数，计数 +1；否则计数 -1
4. 遍历结束后，候选数就是多数元素

关键：摩尔投票算法巧妙地利用多数元素的性质

## Python 代码

```python
# 方法一：哈希表
def majorityElement_hash(nums):
    # 1. 统计每个元素的出现次数
    count_map = {}
    for num in nums:
        count_map[num] = count_map.get(num, 0) + 1
    
    # 2. 找出出现次数大于 n/2 的元素
    target = len(nums) // 2
    for num, count in count_map.items():
        if count > target:
            return num

```

**方法三：摩尔投票算法（最优）**：
```python
def majorityElement(nums):
    # 1. 初始化候选数和计数器
    candidate = None
    count = 0
    
    # 2. 遍历数组
    for num in nums:
        # 3. 计数器为 0，选择当前数为新候选
        if count == 0:
            candidate = num
            count = 1
        elif num == candidate:
            # 4. 当前数等于候选数，计数 +1
            count += 1
        else:
            # 5. 当前数不等于候选数，计数 -1
            count -= 1
    
    # 6. 返回候选数（由于题目保证多数元素存在，候选数一定是多数元素）
    return candidate

```

**方法二：排序**：
```python
def majorityElement_sort(nums):
    # 1. 排序后，中间位置的元素一定是多数元素
    nums.sort()
    return nums[len(nums) // 2]
```

## 易错点
1. **多数元素的定义**：出现次数 > n/2，不是 >= n/2
2. **摩尔投票的核心**：两个不同的数相遇时"相消"，最后剩下的必是多数元素
3. **计数器的初始化**：设为 0，以便第一个数能被正确选择
4. **候选数的更新**：只在计数器为 0 时才更新
5. **题目保证**：题目保证多数元素一定存在，所以无需验证候选数
