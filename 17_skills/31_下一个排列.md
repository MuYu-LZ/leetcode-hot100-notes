# 31. 下一个排列

## 题目
实现获取 **下一个排列** 的函数，算法需要将给定数字序列修改成字典序中的下一个更大的排列。

如果不存在下一个更大的排列，请将数字重新排列成最小的排列（升序）。

必须 **原地修改** ，只允许使用额外常数空间。

示例:
- 输入: nums = [1,2,3]
  输出: [1,3,2]
  
- 输入: nums = [3,2,1]
  输出: [1,2,3]
  
- 输入: nums = [1,1,5]
  输出: [1,5,1]

## 思路
算法步骤：
1. **从右往左**找第一个下降的位置 i（nums[i] < nums[i+1]）
   - 如果不存在，说明已是最大排列，反转整个数组并返回
2. **从右往左**找第一个大于 nums[i] 的元素位置 j（nums[j] > nums[i]）
3. **交换** nums[i] 和 nums[j]
4. **反转** [i+1, n) 的部分，使其成为最小排列

关键：利用从后往前的递减序列特性，通过交换和反转生成下一个排列

## Python 代码

```python
def nextPermutation(nums):
    # 1. 初始化指针
    n = len(nums)
    
    # 2. 从右往左找第一个下降的位置 i
    i = n - 2
    while i >= 0 and nums[i] >= nums[i + 1]:
        i -= 1
    
    # 3. 如果不存在下降位置，说明已是最大排列
    if i == -1:
        # 4. 反转整个数组，得到最小排列
        nums.reverse()
        return
    
    # 5. 从右往左找第一个大于 nums[i] 的元素位置 j
    j = n - 1
    while j > i and nums[j] <= nums[i]:
        j -= 1
    
    # 6. 交换 nums[i] 和 nums[j]
    nums[i], nums[j] = nums[j], nums[i]
    
    # 7. 反转 [i+1, n) 部分
    nums[i + 1:] = reversed(nums[i + 1:])

```

**手工反转版本**（无需 reversed）：
```python
def nextPermutation_manual(nums):
    # 1. 初始化指针
    n = len(nums)
    
    # 2. 从右往左找第一个下降的位置 i
    i = n - 2
    while i >= 0 and nums[i] >= nums[i + 1]:
        i -= 1
    
    # 3. 如果存在下降位置
    if i != -1:
        # 4. 从右往左找第一个大于 nums[i] 的元素位置 j
        j = n - 1
        while j > i and nums[j] <= nums[i]:
            j -= 1
        
        # 5. 交换
        nums[i], nums[j] = nums[j], nums[i]
    
    # 6. 反转 [i+1, n) 部分（无论是否交换）
    left, right = i + 1, n - 1
    while left < right:
        nums[left], nums[right] = nums[right], nums[left]
        left += 1
        right -= 1
```

## 易错点
1. **找下降位置的方向**：必须从右往左，找第一个 nums[i] < nums[i+1]
2. **找交换元素的方向**：也是从右往左，找第一个 nums[j] > nums[i]
3. **边界检查**：i = -1 表示已是最大排列，需要特殊处理
4. **反转范围**：反转 [i+1, n)，即从 i+1 到末尾
5. **原地修改**：直接修改 nums，不返回新列表
