# 75. 颜色分类

## 题目
给定一个包含红、白、蓝三种颜色的数组（分别用 0、1、2 表示），使用原地交换的方法将数组排序，使得相同颜色的元素相邻，并按照红、白、蓝的顺序排列。

必须原地修改数组，且不能使用库中的排序函数。

示例:
- 输入: nums = [2,0,2,1,1,0]
  输出: [0,0,1,1,2,2]
  
- 输入: nums = [2,0,1]
  输出: [0,1,2]
  
- 输入: nums = [1,2,0]
  输出: [0,1,2]

## 思路
使用**三指针**技术（荷兰国旗问题的解法）：
1. 维护三个指针：
   - p0：指向应该放 0 的位置（从左开始）
   - p2：指向应该放 2 的位置（从右开始）
   - curr：当前遍历位置
2. 遍历数组：
   - 如果 nums[curr] == 0，与 nums[p0] 交换，p0++ 和 curr++
   - 如果 nums[curr] == 1，curr++（保留在原地）
   - 如果 nums[curr] == 2，与 nums[p2] 交换，p2--（curr 不动，因为可能交换来 0）
3. 当 curr > p2 时停止

关键：利用三个指针的配合，一次遍历完成排序

## Python 代码

```python
def sortColors(nums):
    # 1. 初始化三个指针
    p0 = 0          # 指向应该放 0 的位置
    p2 = len(nums) - 1  # 指向应该放 2 的位置
    curr = 0        # 当前遍历位置
    
    # 2. 遍历数组
    while curr <= p2:
        # 3. 如果是 0，放到左边
        if nums[curr] == 0:
            nums[p0], nums[curr] = nums[curr], nums[p0]
            p0 += 1
            curr += 1
        # 4. 如果是 2，放到右边
        elif nums[curr] == 2:
            nums[curr], nums[p2] = nums[p2], nums[curr]
            p2 -= 1
            # 注意：curr 不动，因为交换来的可能是 0
        # 5. 如果是 1，保留在原地
        else:
            curr += 1

```

**更简洁的写法**：
```python
def sortColors_concise(nums):
    # 1. 初始化指针
    left, right = 0, len(nums) - 1
    i = 0
    
    # 2. 遍历
    while i <= right:
        if nums[i] == 0:
            # 3. 与左指针交换
            nums[left], nums[i] = nums[i], nums[left]
            left += 1
            i += 1
        elif nums[i] == 2:
            # 4. 与右指针交换
            nums[i], nums[right] = nums[right], nums[i]
            right -= 1
        else:
            # 5. 是 1，直接跳过
            i += 1
```

## 易错点
1. **三个指针的初始化**：p0 从 0 开始，p2 从 n-1 开始，curr 从 0 开始
2. **交换后的指针更新**：
   - 与 p0 交换时，两个指针都要前进
   - 与 p2 交换时，只有 p2 后退，curr 不动
3. **循环条件**：curr <= p2，当 curr > p2 时停止
4. **为什么 curr 与 p2 交换时不前进**：因为交换来的数可能是 0，还需要进一步处理
5. **原地修改**：直接修改 nums，不返回新列表
