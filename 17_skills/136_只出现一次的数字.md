# 136. 只出现一次的数字

## 题目
给定一个非空整数数组，除了某个元素只出现一次外，其余每个元素均出现两次。找出那个只出现一次的元素。

你的算法应该具有线性时间复杂度。你能否仅使用常数空间复杂度来实现？

示例:
- 输入: nums = [2,2,1]
  输出: 1
  
- 输入: nums = [4,1,2,1,2]
  输出: 4
  
- 输入: nums = [1]
  输出: 1

## 思路
使用**异或操作（XOR）**的性质：
1. a ^ a = 0（任何数与自己异或为 0）
2. a ^ 0 = a（任何数与 0 异或为自己）
3. 异或满足交换律和结合律
4. 因此，所有数异或起来，相同的数会消除，剩下的就是只出现一次的数
5. 时间复杂度：O(n)，空间复杂度：O(1)

关键：利用异或的特殊性质，无需额外空间即可找到答案

## Python 代码

```python
def singleNumber(nums):
    # 1. 初始化异或结果为 0
    result = 0
    
    # 2. 遍历数组中的每个数
    for num in nums:
        # 3. 将所有数进行异或操作
        result ^= num
    
    # 4. 剩下的就是只出现一次的数
    return result

```

**一行代码版本**：
```python
from functools import reduce
import operator

def singleNumber_oneline(nums):
    return reduce(operator.xor, nums, 0)
```

## 易错点
1. **异或的性质**：必须理解 a ^ a = 0 和 a ^ 0 = a
2. **交换律**：异或操作可以任意排列，不需要按特定顺序
3. **结合律**：(a ^ b) ^ c = a ^ (b ^ c)
4. **初始值**：从 0 开始异或，因为 a ^ 0 = a
5. **时间/空间复杂度**：O(n) 时间，O(1) 空间，满足题目要求
