# 84. 柱状图中最大的矩形

## 题目
给定 n 个非负整数，用来表示柱状图中各个柱子的高度。这些柱子紧靠在一起形成一个柱状图，计算柱状图中能够容纳的最大矩形的面积。

示例:
- 输入: heights = [2,1,5,6,2,3]
  输出: 10
  解释: 最大矩形面积为 10（[5,6] 形成的矩形，宽度为 2，高度为 5）
  
- 输入: heights = [2,4]
  输出: 4
  
- 输入: heights = [0,0]
  输出: 0

## 思路
使用**单调栈**优化：
1. 栈维护索引，保持对应高度单调递增
2. 当遇到更低的柱子时，弹出栈中所有更高的柱子
3. 对于每个被弹出的柱子 h：
   - 宽度为：当前位置到栈顶的距离（i - stack[-1] - 1）
   - 面积为：h * 宽度
4. 添加哨兵（高度 0）在开始和结束，确保所有柱子都被处理

关键：每个柱子作为高度时，其左右边界由栈确定

## Python 代码

```python
def largestRectangleArea(heights):
    # 1. 添加哨兵值在两端
    heights = [0] + heights + [0]
    stack = []
    max_area = 0
    
    # 2. 遍历所有柱子
    for i in range(len(heights)):
        # 3. 当前高度低于栈顶时，弹出并计算面积
        while stack and heights[i] < heights[stack[-1]]:
            # 4. 弹出栈顶索引（被弹出柱子作为矩形的高）
            h_idx = stack.pop()
            h = heights[h_idx]
            # 5. 宽度：从栈顶下一个元素到当前位置
            w = i - stack[-1] - 1
            # 6. 计算面积
            max_area = max(max_area, h * w)
        
        # 7. 将当前索引压入栈
        stack.append(i)
    
    return max_area

```

## 易错点
1. **哨兵的作用**：添加首尾的 0 确保所有柱子都能被弹出并计算
2. **宽度计算**：宽度是 i - stack[-1] - 1，不能是 i - h_idx，因为需要找到左边界
3. **栈内单调性**：栈中索引对应的高度保持单调递增（除非弹出）
4. **面积公式**：面积 = 被弹出柱子的高 × 该高度对应的最大宽度
5. **索引管理**：h_idx 是被弹出柱子，i 是当前柱子，stack[-1] 是左边界，要小心这三个的含义
