# 020. 有效的括号

## 题目
给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s，判断字符串是否有效。

有效字符串需满足：
1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。

示例 1:
输入: s = "()"
输出: true

示例 2:
输入: s = "([)]"
输出: false

示例 3:
输入: s = "{[]}"
输出: true

## 思路
该题使用**栈**验证括号的有效性：
1. 遍历字符串中的每个字符
2. 如果是左括号，压入栈
3. 如果是右括号：
   - 检查栈是否为空（没有对应的左括号）
   - 检查栈顶的左括号是否与当前右括号匹配
   - 如果匹配，弹出栈顶；否则返回 False
4. 遍历完成后，栈必须为空（所有左括号都被闭合）

## Python 代码

```python
class Solution:
    def isValid(self, s: str) -> bool:
        # 1. 括号映射：右括号 -> 左括号
        mapping = {')': '(', '}': '{', ']': '['}
        stack = []
        
        # 2. 遍历字符串
        for char in s:
            if char in mapping:
                # 3. 当前字符是右括号
                # 检查栈是否为空或栈顶不匹配
                if not stack or stack[-1] != mapping[char]:
                    return False
                # 4. 匹配成功，弹出栈顶
                stack.pop()
            else:
                # 5. 当前字符是左括号，压入栈
                stack.append(char)
        
        # 6. 最后栈必须为空
        return len(stack) == 0

```

## 易错点
1. **括号映射的方向**：使用右括号 -> 左括号的映射，便于检查匹配
2. **栈为空的检查**：必须先检查栈是否为空，再访问 stack[-1]
3. **顺序很重要**：{} 和 }{ 的顺序不同，必须匹配
4. **最后的栈检查**：遍历完成后栈必须为空，否则有未闭合的左括号
