# 394. 字符串解码

## 题目
给定一个经过编码的字符串，返回它解码后的字符串。编码规则为：k[encoded_string]，表示其中方括号内部的 encoded_string 正好重复 k 次。注意 k 保证为正整数。

你可以认为输入字符串总是有效的；输入字符串中没有额外的空格，且输入的方括号总是符合格式的。

示例:
- 输入: s = "3[a2[c]]"
  输出: "accaccacc"
  
- 输入: s = "2[abc]3[cd]edt"
  输出: "abcabccdcdcdedt"
  
- 输入: s = "a2[2[y]pq4[2[jk]e1[z]]]2[abc]"
  输出: "ayypqjkjkezyypqjkjkezyypqjkjkezayypqjkjkezyypqjkjkezabcabc"

## 思路
使用**栈**处理嵌套的编码：
1. 遍历字符串
2. 遇到数字则累加数字（处理多位数字）
3. 遇到字母则添加到当前字符串
4. 遇到 `[` 则：将当前数字和字符串压入栈
5. 遇到 `]` 则：弹出数字和之前的字符串，进行重复操作
6. 返回最终结果

关键点：栈中存储 (前置字符串, 重复次数) 元组，处理嵌套结构

## Python 代码

```python
def decodeString(s: str) -> str:
    # 1. 使用栈存储 (字符串, 数字) 对
    stack = []
    current_str = ""
    current_num = ""
    
    # 2. 遍历每个字符
    for char in s:
        if char.isdigit():
            # 3. 累加数字
            current_num += char
        elif char == '[':
            # 4. 压入栈：之前的字符串和数字
            stack.append((current_str, int(current_num)))
            current_str = ""
            current_num = ""
        elif char == ']':
            # 5. 弹出栈，进行重复操作
            prev_str, num = stack.pop()
            current_str = prev_str + num * current_str
        else:
            # 6. 普通字母直接添加
            current_str += char
    
    return current_str

```

## 易错点
1. **多位数字处理**：数字可能是多位，需要连续累加而不是单个字符
2. **栈的存储结构**：每次压入时需要保存 (当前字符串, 当前数字) 的状态
3. **栈弹出后的操作**：弹出后应该是前置字符串 + num 次当前字符串
4. **重置变量**：每次 `[` 后需要重置 current_str 和 current_num
5. **嵌套处理**：栈的多层嵌套保证了内层先处理，然后返回到外层
