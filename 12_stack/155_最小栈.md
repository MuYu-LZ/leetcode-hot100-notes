# 155. 最小栈

## 题目
设计一个支持 push、pop、top 操作，并能在常数时间内检索到最小元素的栈。

实现 MinStack 类:
- MinStack() 初始化栈对象。
- void push(int val) 将元素val推入栈中。
- void pop() 删除栈顶的元素。
- int top() 获取栈顶的元素。
- int getMin() 返回栈中的最小元素。

示例:
输入:
["MinStack","push","push","push","getMin","pop","top","getMin"]
[[],[0],[1],[0],[],[],[],[]]
输出:
[null,null,null,null,0,null,0,0]

## 思路
该题使用**两个栈**维护最小值：
1. 一个主栈存储所有元素
2. 一个最小栈存储对应位置的最小值
3. push：同时向两个栈添加元素，最小栈添加当前最小值（min(val, min_stack.top())）
4. pop：同时从两个栈弹出元素
5. getMin：返回最小栈的栈顶元素，O(1) 时间复杂度

## Python 代码

```python
class MinStack:
    def __init__(self):
        # 1. 主栈和最小栈
        self.stack = []
        self.min_stack = []
    
    def push(self, val: int) -> None:
        # 2. 主栈添加元素
        self.stack.append(val)
        
        # 3. 最小栈添加当前最小值
        if not self.min_stack:
            self.min_stack.append(val)
        else:
            self.min_stack.append(min(val, self.min_stack[-1]))
    
    def pop(self) -> None:
        # 4. 同时从两个栈弹出
        self.stack.pop()
        self.min_stack.pop()
    
    def top(self) -> int:
        # 5. 返回主栈的栈顶
        return self.stack[-1]
    
    def getMin(self) -> int:
        # 6. 返回最小栈的栈顶
        return self.min_stack[-1]

```

## 易错点
1. **两个栈的同步**：push 和 pop 时必须同时操作两个栈，保持同步
2. **最小栈的维护**：push 时添加当前的最小值，即 min(val, min_stack[-1])
3. **初始化**：最小栈为空时，第一个元素就是最小值
4. **时间复杂度**：所有操作都是 O(1)
5. **空间复杂度**：O(n)，两个栈各占用 O(n) 空间
