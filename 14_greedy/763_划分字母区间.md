# 763. 划分字母区间

## 题目
给你一个字符串 s 。我们要把这个字符串划分为尽可能多的片段，同一字母最多出现在一个片段中。

返回一个表示每个字符串片段长度的列表。

示例:
- 输入: s = "ababcbacaddefegdehijhij"
  输出: [9,7,8]
  解释: 划分结果为 "ababcbaca", "defegde", "hijhij"。每个字母最多出现在一个片段中，没有字母出现在多个片段中
  
- 输入: s = "abc"
  输出: [3]
  
- 输入: s = "abcd"
  输出: [1,1,1,1]

## 思路
使用**贪心 + 哈希表**：
1. 第一次遍历：记录每个字符最后出现的位置（last_occurrence）
2. 第二次遍历，维护当前片段的最远边界 (max_end)：
   - 对于每个字符，更新 max_end = max(max_end, last_occurrence[char])
   - 当到达 max_end 时，说明可以形成一个片段，计算长度并添加
3. 贪心策略：尽早确定片段边界，每当能确定边界时立即划分

关键：利用字符最后出现位置，确定片段的最远边界

## Python 代码

```python
def partitionLabels(s):
    # 1. 第一次遍历：记录每个字符最后出现的位置
    last_occurrence = {}
    for i, char in enumerate(s):
        last_occurrence[char] = i
    
    # 2. 初始化结果列表和指针
    result = []
    start = 0
    max_end = 0
    
    # 3. 第二次遍历：确定片段边界
    for i, char in enumerate(s):
        # 4. 更新当前片段的最远边界
        max_end = max(max_end, last_occurrence[char])
        
        # 5. 当到达最远边界时，形成一个片段
        if i == max_end:
            # 6. 计算片段长度
            result.append(i - start + 1)
            # 7. 更新下一个片段的起始位置
            start = i + 1
    
    return result

```

## 易错点
1. **两次遍历的目的**：第一次收集信息，第二次利用信息进行划分
2. **最后出现位置的记录**：必须记录每个字符最后一次出现的位置，而非第一次
3. **片段边界的判定**：当到达 max_end 时才能确定片段，因为可能有其他字符的最后出现位置在 max_end
4. **片段长度计算**：长度为 i - start + 1（包括两端）
5. **起始位置的更新**：划分后，start 应该更新为 i + 1
