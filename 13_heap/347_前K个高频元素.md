# 347. 前K个高频元素

## 题目
给你一个整数数组 nums 和一个整数 k ，请你返回其中出现频率前 k 高的元素。你可以按任意顺序返回答案。

示例:
- 输入: nums = [1,1,1,2,2,3], k = 2
  输出: [1,2]
  
- 输入: nums = [4,1,1,1,2,2,2], k = 2
  输出: [2,1]
  
- 输入: nums = [1], k = 1
  输出: [1]

## 思路
使用**最小堆**维护前 k 个频率最高的元素：
1. 统计所有元素的出现频率（使用哈希表）
2. 维护一个大小为 k 的最小堆，按频率排序
3. 遍历所有不同元素：
   - 如果堆大小 < k，直接添加元素
   - 否则，如果当前元素频率 > 堆顶频率，删除堆顶，添加当前元素
4. 最后堆中的 k 个元素就是答案
5. 时间复杂度：O(n log k)

关键：最小堆保持 k 个频率最高的元素，堆顶是其中频率最低的

## Python 代码

```python
import heapq
from collections import Counter

def topKFrequent(nums, k):
    # 1. 统计元素频率
    freq = Counter(nums)
    
    # 2. 创建最小堆，存储 (频率, 元素) 对
    heap = []
    
    # 3. 遍历所有不同的元素
    for num, count in freq.items():
        if len(heap) < k:
            # 4. 堆未满，直接添加
            heapq.heappush(heap, (count, num))
        elif count > heap[0][0]:
            # 5. 堆已满且当前频率比堆顶大
            # 6. 删除堆顶，添加当前元素
            heapq.heapreplace(heap, (count, num))
    
    # 7. 提取堆中所有元素
    return [num for count, num in heap]

```

## 易错点
1. **计数器的使用**：Counter 可以直接统计频率，比手动遍历更简洁
2. **堆存储结构**：堆中存储 (频率, 元素) 元组，按频率比较
3. **最小堆维护**：堆保持大小 k，确保堆顶是前 k 个中频率最低的
4. **比较条件**：只有当新元素的频率**严格大于**堆顶频率时才替换
5. **结果顺序**：题目允许任意顺序，不需要额外排序，直接从堆中提取即可
