# 295. 数据流的中位数

## 题目
中位数是有序整数列表中的中间值。如果列表的大小是偶数，则没有中间值，中位数是两个中间值的平均数。

实现 MedianFinder 类:
- MedianFinder() 初始化 MedianFinder 对象。
- void addNum(int num) 从数据流中添加一个整数到数据结构中。
- double findMedian() 返回到目前为止所有添加的数中的中位数。

示例:
输入:
["MedianFinder", "addNum", "findMedian", "addNum", "findMedian", "addNum", "findMedian"]
[[], [1], [], [2], [], [3], []]
输出:
[null, null, 1.0, null, 1.5, null, 2.0]

## 思路
使用**两个堆**维护数据流的中位数：
1. 大顶堆（用最小堆模拟）：存储较小的一半元素
2. 小顶堆：存储较大的一半元素
3. 维持平衡：|大顶堆| == |小顶堆| 或 |大顶堆| == |小顶堆| + 1
4. 添加元素时：
   - 先加入大顶堆
   - 如果大顶堆最大值 > 小顶堆最小值，则交换堆顶
   - 如果大顶堆大小 > 小顶堆大小 + 1，则移动一个元素
5. 求中位数：取决于是否平衡

关键：大顶堆存小的一半，小顶堆存大的一半，两堆大小相近

## Python 代码

```python
import heapq

class MedianFinder:
    def __init__(self):
        # 1. 大顶堆（用负数模拟）存较小的一半
        self.small = []
        # 2. 小顶堆存较大的一半
        self.large = []
    
    def addNum(self, num: int) -> None:
        # 3. 先加入大顶堆（负数）
        heapq.heappush(self.small, -num)
        
        # 4. 保证小顶堆的最小值 >= 大顶堆的最大值
        if self.small and self.large and (-self.small[0] > self.large[0]):
            # 5. 交换两堆的顶部元素
            val = -heapq.heappop(self.small)
            heapq.heappush(self.large, val)
        
        # 6. 保持大小平衡：大顶堆最多比小顶堆多 1 个元素
        if len(self.small) > len(self.large) + 1:
            val = -heapq.heappop(self.small)
            heapq.heappush(self.large, val)
        
        # 7. 保持小顶堆不小于大顶堆
        if len(self.large) > len(self.small):
            val = heapq.heappop(self.large)
            heapq.heappush(self.small, -val)
    
    def findMedian(self) -> float:
        # 8. 如果元素数是奇数
        if len(self.small) > len(self.large):
            return float(-self.small[0])
        # 9. 如果元素数是偶数，返回两堆顶的平均值
        return (-self.small[0] + self.large[0]) / 2.0

```

## 易错点
1. **大顶堆的模拟**：Python 的 heapq 是最小堆，用负数来模拟最大堆
2. **堆大小平衡**：需要多次检查和调整以保证大小关系正确
3. **堆顶关系维护**：大顶堆的最大值 <= 小顶堆的最小值
4. **中位数计算**：奇数个元素时取大顶堆堆顶，偶数个元素时取两堆堆顶的平均值
5. **添加元素的顺序**：先加大顶堆，再做平衡调整，最后再做大小调整
