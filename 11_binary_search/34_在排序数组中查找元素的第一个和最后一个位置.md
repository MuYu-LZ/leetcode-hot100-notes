# 34. 在排序数组中查找元素的第一个和最后一个位置

## 题目
给你一个按照非递减顺序排列的整数数组 nums，和一个整数 target 。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 target，返回 [-1, -1]。

你必须设计并实现时间复杂度为 O(log n) 的算法解决此问题。

示例:
- 输入: nums = [5,7,7,8,8,10], target = 8
  输出: [3,4]
  
- 输入: nums = [5,7,7,8,8,10], target = 6
  输出: [-1,-1]
  
- 输入: nums = [], target = 0
  输出: [-1,-1]

## 思路
使用**两次二分查找**分别找起始和结束位置：
1. **第一次二分查找**：找最左边的 target（第一个出现位置）
   - 当 nums[mid] >= target 时，继续在左侧查找，收缩右边界
   - 找到范围边界，而不是停在第一个相等处
2. **第二次二分查找**：找最右边的 target（最后一个出现位置）
   - 当 nums[mid] <= target 时，继续在右侧查找，扩大左边界
3. 检查边界条件，防止越界或目标不存在的情况

关键：利用二分查找找边界的特性，两次查找分别确定左右边界

## Python 代码

```python
def searchRange(nums, target):
    # 1. 定义二分查找找最左边界的函数
    def findFirst(nums, target):
        left, right = 0, len(nums) - 1
        result = -1
        
        # 2. 查找最左边的 target
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] == target:
                # 3. 找到了，但继续在左侧寻找更左的
                result = mid
                right = mid - 1
            elif nums[mid] < target:
                # 4. 目标在右侧
                left = mid + 1
            else:
                # 5. 目标在左侧
                right = mid - 1
        
        return result
    
    # 6. 定义二分查找找最右边界的函数
    def findLast(nums, target):
        left, right = 0, len(nums) - 1
        result = -1
        
        # 7. 查找最右边的 target
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] == target:
                # 8. 找到了，但继续在右侧寻找更右的
                result = mid
                left = mid + 1
            elif nums[mid] < target:
                # 9. 目标在右侧
                left = mid + 1
            else:
                # 10. 目标在左侧
                right = mid - 1
        
        return result
    
    # 11. 如果数组为空，直接返回 [-1, -1]
    if not nums:
        return [-1, -1]
    
    # 12. 调用两个查找函数
    first = findFirst(nums, target)
    
    # 13. 如果第一个位置不存在，返回 [-1, -1]
    if first == -1:
        return [-1, -1]
    
    last = findLast(nums, target)
    return [first, last]

```

## 易错点
1. **两个查找的区别**：第一个查找在找到时收缩右边界，第二个查找在找到时扩大左边界
2. **继续查找的必要性**：找到目标后不能立即返回，要继续向该方向查找
3. **边界检查**：必须检查 first == -1，如果不存在任何目标，不需要查找 last
4. **空数组处理**：提前检查数组是否为空，避免后续索引错误
5. **返回值类型**：一定要返回列表 [first, last]，即使都不存在也是 [-1, -1]
