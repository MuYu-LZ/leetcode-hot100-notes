# 4. 寻找两个正序数组的中位数

## 题目
给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2。请你找出并返回这两个正序数组的 中位数 。

算法的时间复杂度应该为 O(log (m+n)) 。

示例:
- 输入: nums1 = [1,3], nums2 = [2]
  输出: 2.00000
  解释: 合并后的数组为 [1,2,3]，中位数是 2
  
- 输入: nums1 = [1,2], nums2 = [3,4]
  输出: 2.50000
  解释: 合并后的数组为 [1,2,3,4]，中位数是 (2 + 3) / 2 = 2.5
  
- 输入: nums1 = [0,0], nums2 = [0,0]
  输出: 0.00000

## 思路
使用**二分查找**在较短的数组上进行：
1. 为了达到 O(log(m+n))，需要在较短数组上二分
2. 分割两个数组，使得左侧元素个数 = 右侧元素个数
3. 关键条件：nums1[cut1 - 1] <= nums2[cut2] 且 nums2[cut2 - 1] <= nums1[cut1]
4. 在满足条件的分割点计算中位数：
   - 偶数个元素：(左侧最大值 + 右侧最小值) / 2
   - 奇数个元素：左侧最大值
5. 时间复杂度：O(log(min(m, n)))

关键：分割使得两侧平衡，然后对调整分割点进行二分

## Python 代码

```python
def findMedianSortedArrays(nums1, nums2):
    # 1. 确保 nums1 是较短的数组，减少二分次数
    if len(nums1) > len(nums2):
        return findMedianSortedArrays(nums2, nums1)
    
    m, n = len(nums1), len(nums2)
    # 2. 左侧应该有 (m + n + 1) // 2 个元素
    half = (m + n + 1) // 2
    
    left, right = 0, m
    
    # 3. 在 nums1 上进行二分查找
    while left <= right:
        cut1 = (left + right) // 2
        cut2 = half - cut1
        
        # 4. 处理边界情况
        left1 = float('-inf') if cut1 == 0 else nums1[cut1 - 1]
        right1 = float('inf') if cut1 == m else nums1[cut1]
        left2 = float('-inf') if cut2 == 0 else nums2[cut2 - 1]
        right2 = float('inf') if cut2 == n else nums2[cut2]
        
        # 5. 检查分割是否满足条件
        if left1 <= right2 and left2 <= right1:
            # 6. 满足条件，计算中位数
            if (m + n) % 2 == 0:
                # 7. 偶数个元素，返回两个中间值的平均
                return (max(left1, left2) + min(right1, right2)) / 2.0
            else:
                # 8. 奇数个元素，返回左侧最大值
                return float(max(left1, left2))
        elif left1 > right2:
            # 9. cut1 太大，向左移动
            right = cut1 - 1
        else:
            # 10. cut1 太小，向右移动
            left = cut1 + 1

```

## 易错点
1. **确保较短数组优先**：避免对较长数组进行二分，减少时间复杂度
2. **半长度计算**：(m + n + 1) // 2 处理奇偶数差异，确保左侧和右侧平衡
3. **边界值设置**：使用 float('-inf') 和 float('inf') 处理分割点在数组端点的情况
4. **分割条件**：left1 <= right2 且 left2 <= right1 必须同时满足
5. **奇偶数处理**：奇数个元素只需取左侧最大，偶数个元素取两侧平均
