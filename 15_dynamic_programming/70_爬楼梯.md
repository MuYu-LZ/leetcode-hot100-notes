# 70. 爬楼梯

## 题目
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 个或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

示例:
- 输入: n = 2
  输出: 2
  解释: 有两种方法可以爬到楼顶。1. 1 阶 + 1 阶；2. 2 阶
  
- 输入: n = 3
  输出: 3
  解释: 有三种方法可以爬到楼顶。1. 1 阶 + 1 阶 + 1 阶；2. 1 阶 + 2 阶；3. 2 阶 + 1 阶
  
- 输入: n = 4
  输出: 5

## 思路
使用**动态规划**，递推关系为 dp[i] = dp[i-1] + dp[i-2]：
1. dp[i] 表示到达第 i 阶的方法数
2. 第 i 阶可以从第 i-1 阶爬 1 个台阶到达，也可以从第 i-2 阶爬 2 个台阶到达
3. 因此 dp[i] = dp[i-1] + dp[i-2]，这是一个斐波那契数列
4. 初始条件：dp[0] = 1（停留在原地），dp[1] = 1（爬 1 阶）
5. 优化：只需维护前两个状态，空间复杂度可优化为 O(1)

关键：识别递推关系，每个状态只依赖于前两个状态

## Python 代码

```python
def climbStairs(n):
    # 1. 特殊情况处理
    if n <= 1:
        return 1
    
    # 2. 初始化 dp 数组
    dp = [0] * (n + 1)
    dp[0] = 1
    dp[1] = 1
    
    # 3. 从第 2 阶开始填充 dp 数组
    for i in range(2, n + 1):
        # 4. 第 i 阶的方法数 = 从第 i-1 阶爬来 + 从第 i-2 阶爬来
        dp[i] = dp[i - 1] + dp[i - 2]
    
    return dp[n]

```

**空间优化版本**：
```python
def climbStairs_optimize(n):
    # 1. 只维护前两个状态
    if n <= 1:
        return 1
    
    prev1 = 1  # dp[i-1]
    prev2 = 1  # dp[i-2]
    
    # 2. 从第 2 阶开始计算
    for i in range(2, n + 1):
        # 3. 当前阶的方法数
        curr = prev1 + prev2
        # 4. 更新前两个状态
        prev2 = prev1
        prev1 = curr
    
    return prev1
```

## 易错点
1. **基础情况**：dp[0] = 1（可以不动），dp[1] = 1（爬 1 阶）
2. **递推公式**：dp[i] 只依赖于 dp[i-1] 和 dp[i-2]
3. **循环起点**：从 i=2 开始，因为 dp[0] 和 dp[1] 已初始化
4. **数组大小**：需要 n+1 个位置来存储 0 到 n 的值
5. **空间优化**：若空间是瓶颈，可以只用两个变量维护前两个状态
