# 279. 完全平方数

## 题目
给你一个整数 n ，返回 **和为 n 的完全平方数的最少数量** 。

完全平方数 是一个整数，其值等于另一个整数的平方；换句话说，其值等于某个整数的平方。例如，1、4、9 和 16 是完全平方数，而 3 和 11 不是。

示例:
- 输入: n = 7
  输出: 2
  解释: 7 = 4 + 3，即 7 = 2² + 3（无法表示为更少完全平方数的和）
  
- 输入: n = 2
  输出: 2
  解释: 2 = 1 + 1
  
- 输入: n = 13
  输出: 2
  解释: 13 = 4 + 9 = 2² + 3²

## 思路
使用**动态规划**，类似完全背包问题：
1. dp[i] 表示和为 i 的完全平方数的最少数量
2. 对于每个数 i，遍历所有小于等于 i 的完全平方数 j²
3. 递推关系：dp[i] = min(dp[i], dp[i - j²] + 1)
4. 初始条件：dp[0] = 0（0 不需要任何完全平方数）
5. 遍历顺序：外层 i，内层 j（完全背包）

关键：完全背包问题，每个完全平方数可以重复使用

## Python 代码

```python
def numSquares(n):
    # 1. 初始化 dp 数组，dp[i] = 和为 i 的最少完全平方数个数
    dp = [float('inf')] * (n + 1)
    dp[0] = 0  # 2. 初始条件：0 需要 0 个
    
    # 3. 遍历从 1 到 n 的每个数
    for i in range(1, n + 1):
        # 4. 遍历所有小于等于 i 的完全平方数
        j = 1
        while j * j <= i:
            # 5. 尝试用完全平方数 j² 组成 i
            dp[i] = min(dp[i], dp[i - j * j] + 1)
            j += 1
    
    return dp[n]

```

**优化版本（数学方法）**：
根据四平方和定理，任何正整数都可以表示为最多 4 个完全平方数的和。若 n 是完全平方数，答案为 1；若 n 可以表示为两个完全平方数之和，答案为 2。

```python
def numSquares_optimize(n):
    # 1. 检查是否是完全平方数
    if int(n ** 0.5) ** 2 == n:
        return 1
    
    # 2. 检查是否是两个完全平方数之和
    i = 1
    while i * i <= n:
        if int((n - i * i) ** 0.5) ** 2 == n - i * i:
            return 2
        i += 1
    
    # 3. 检查是否能表示为 4k + 3 的形式（若是，则需要 4 个）
    while n % 4 == 0:
        n //= 4
    if n % 8 == 7:
        return 4
    
    return 3
```

## 易错点
1. **初始化**：dp[0] = 0，其他元素初始化为无穷大
2. **完全平方数的生成**：使用 j*j <= i 作为循环条件，避免超出范围
3. **递推关系**：dp[i] = min(dp[i], dp[i - j*j] + 1)
4. **遍历顺序**：外层 i 内层 j 是正确的顺序（完全背包）
5. **时间复杂度**：DP 版本为 O(n * sqrt(n))，优化版本为 O(sqrt(n))
