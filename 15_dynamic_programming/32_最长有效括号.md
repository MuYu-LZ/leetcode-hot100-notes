# 32. 最长有效括号

## 题目
给你一个只包含 '(' 和 ')' 的字符串，找出最长有效（格式正确且连续）括号子串的长度。

示例:
- 输入: s = "(()"
  输出: 2
  解释: 最长有效括号子串是 "()"，长度为 2
  
- 输入: s = ")()())"
  输出: 4
  解释: 最长有效括号子串是 "()()"，长度为 4
  
- 输入: s = ""
  输出: 0

## 思路
**方法一：动态规划**（O(n)）：
1. dp[i] 表示以位置 i 结尾的最长有效括号长度
2. 如果 s[i] == '('，则 dp[i] = 0（'(' 不能结尾）
3. 如果 s[i] == ')'：
   - 如果 s[i-1] == '('，则 dp[i] = dp[i-2] + 2（配对成功）
   - 如果 s[i-1] == ')' 且 s[i - dp[i-1] - 1] == '('，则 dp[i] = dp[i-1] + 2 + dp[i - dp[i-1] - 2]
4. 答案为 max(dp)

**方法二：栈方法**：
维护栈底始终有一个不匹配的 ')' 的索引，简化匹配逻辑。

关键：DP 的递推关系比较复杂，需要仔细处理配对情况

## Python 代码

```python
def longestValidParentheses(s):
    # 1. 初始化 dp 数组，dp[i] = 以位置 i 结尾的最长有效括号长度
    n = len(s)
    if n < 2:
        return 0
    
    dp = [0] * n
    max_len = 0
    
    # 2. 从位置 1 开始遍历
    for i in range(1, n):
        # 3. 如果当前是 ')'
        if s[i] == ')':
            # 4. 情况 1：前一个是 '('，形成配对 "()"
            if s[i - 1] == '(':
                # 5. 加上前面的有效长度
                dp[i] = (dp[i - 2] if i >= 2 else 0) + 2
            # 6. 情况 2：前一个是 ')'
            elif i - dp[i - 1] > 0 and s[i - dp[i - 1] - 1] == '(':
                # 7. 当前 ')' 与前面某个 '(' 配对
                # 8. 加上前面的有效长度和之前更前面的有效长度
                dp[i] = dp[i - 1] + 2 + (dp[i - dp[i - 1] - 2] if i - dp[i - 1] >= 2 else 0)
        
        # 9. 更新全局最大值
        max_len = max(max_len, dp[i])
    
    return max_len

```

**栈方法**：
```python
def longestValidParentheses_stack(s):
    # 1. 栈底初始化为 -1（作为参考点）
    stack = [-1]
    max_len = 0
    
    # 2. 遍历每个字符
    for i in range(len(s)):
        if s[i] == '(':
            # 3. '(' 直接压入栈
            stack.append(i)
        else:  # s[i] == ')'
            # 4. ')' 弹出栈顶
            stack.pop()
            
            if not stack:
                # 5. 栈空，当前 ')' 不匹配，作为新的参考点
                stack.append(i)
            else:
                # 6. 计算当前有效长度
                max_len = max(max_len, i - stack[-1])
    
    return max_len
```

## 易错点
1. **DP 的索引检查**：访问 dp[i-2] 前需要检查 i >= 2
2. **情况 2 的判断**：需要检查 i - dp[i-1] > 0，确保有可配对的 '('
3. **累加长度**：需要加上前面的有效长度，而不仅仅是当前的配对
4. **栈方法的参考点**：栈底始终保留一个不匹配的索引，简化计算
5. **初始化**：DP 初始化为 0，最大值初始化为 0
