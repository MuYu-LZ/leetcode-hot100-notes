# 139. 单词拆分

## 题目
给你一个字符串 s 和一个字符串列表 wordDict 作为字典。请你判断是否可以利用字典中出现的单词拼接出 s 。

注意：同一个字典中的单词可以在分段中被重复使用。

示例:
- 输入: s = "leetcode", wordDict = ["leet","code"]
  输出: true
  
- 输入: s = "applepenapple", wordDict = ["apple","pen"]
  输出: true
  
- 输入: s = "catsandog", wordDict = ["cat","cats","and","sand","dog"]
  输出: false

## 思路
使用**动态规划**：
1. dp[i] 表示字符串 s 的前 i 个字符是否能被字典单词拼接成
2. 对于每个位置 i，检查所有以 i 结尾的单词：
   - 如果 s[j:i] 在字典中，且 dp[j] 为 true，则 dp[i] = true
3. 递推关系：dp[i] = dp[j] and s[j:i] in wordDict
4. 初始条件：dp[0] = true（空字符串可以被拼接）
5. 优化：用集合存储字典，使查询为 O(1)

关键：遍历所有可能的前一个位置 j，检查是否能形成有效的单词

## Python 代码

```python
def wordBreak(s, wordDict):
    # 1. 将字典转换为集合，加快查询速度
    word_set = set(wordDict)
    
    # 2. 初始化 dp 数组，dp[i] = s 的前 i 个字符是否能被拼接
    dp = [False] * (len(s) + 1)
    dp[0] = True  # 3. 空字符串可以被拼接
    
    # 4. 遍历字符串的每个位置
    for i in range(1, len(s) + 1):
        # 5. 检查所有以 i 结尾的可能单词
        for j in range(i):
            # 6. 如果 s[j:i] 在字典中，且 dp[j] 为 true
            if dp[j] and s[j:i] in word_set:
                # 7. 则 dp[i] = true
                dp[i] = True
                break  # 8. 提前退出，优化性能
    
    return dp[len(s)]

```

## 易错点
1. **集合的使用**：将列表转换为集合，使 in 操作为 O(1)
2. **初始化**：dp[0] = true，表示空字符串可以被拼接
3. **子串的提取**：s[j:i] 表示从位置 j（不含）到位置 i（含）的子串
4. **提前退出**：找到一个有效的 j 后可以 break，因为只需 dp[i] = true
5. **时间复杂度**：O(n²) 位置，每个位置最多检查 n 个前置位置
