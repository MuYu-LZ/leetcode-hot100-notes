# 416. 分割等和子集

## 题目
给你一个 **只包含正整数** 的非空数组 nums 。请你判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

示例:
- 输入: nums = [1,5,11,5]
  输出: true
  解释: 数组可以分割成 [1, 5, 5] 和 [11]。
  
- 输入: nums = [2,2,1,1]
  输出: true
  
- 输入: nums = [1,2,5]
  输出: false

## 思路
转化为**0-1 背包问题**：
1. 计算数组总和 total_sum
2. 如果 total_sum 是奇数，无法分割，返回 false
3. 问题转化为：是否存在子集，其和等于 total_sum / 2
4. 使用 DP：dp[i] 表示是否能凑出和为 i
5. 对于每个数 num，遍历从 target 到 num 的值（0-1 背包逆序遍历）
6. 递推关系：dp[i] = dp[i] or dp[i - num]
7. 答案为 dp[target]

关键：转化为背包问题，每个数字恰好使用一次（0-1 背包）

## Python 代码

```python
def canPartition(nums):
    # 1. 计算数组总和
    total_sum = sum(nums)
    
    # 2. 如果总和是奇数，无法分割
    if total_sum % 2 != 0:
        return False
    
    target = total_sum // 2
    
    # 3. 初始化 dp 数组，dp[i] = 是否能凑出和为 i
    dp = [False] * (target + 1)
    dp[0] = True  # 4. 和为 0 总是可以凑出（什么都不选）
    
    # 5. 遍历每个数
    for num in nums:
        # 6. 逆序遍历，保证每个数只用一次（0-1 背包）
        for i in range(target, num - 1, -1):
            # 7. 更新 dp[i]
            dp[i] = dp[i] or dp[i - num]
    
    return dp[target]

```

## 易错点
1. **总和的奇偶性**：奇数总和无法分割成两个相等的子集
2. **目标值**：target = total_sum // 2
3. **逆序遍历**：0-1 背包需要逆序遍历，防止一个数字被多次使用
4. **初始化**：dp[0] = true（空子集的和为 0）
5. **遍历范围**：从 target 到 num，步长为 -1
