# 198. 打家劫舍

## 题目
你是一个专业的小偷，计划偷窃沿街的房屋。每间房屋都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，**如果两间相邻的房屋在同一晚被偷窃，系统会自动报警**。

给定一个代表每个房屋存放金额的非负整数数组，计算你 **不触动警报装置的情况下** ，一夜之内能够偷窃到的最高金额。

示例:
- 输入: nums = [1,2,3,1]
  输出: 4
  解释: 偷窃 1 号房屋（金额 = 1）和 3 号房屋（金额 = 3）。偷窃到的最高金额 = 1 + 3 = 4
  
- 输入: nums = [2,7,9,3]
  输出: 9
  解释: 偷窃 9 号房屋（金额 = 9）。偷窃到的最高金额 = 9
  
- 输入: nums = [5,2,6,3]
  输出: 11
  解释: 偷窃 1 号房屋（金额 = 5）和 3 号房屋（金额 = 6）。偷窃到的最高金额 = 5 + 6 = 11

## 思路
使用**动态规划**，递推关系为 dp[i] = max(dp[i-1], dp[i-2] + nums[i])：
1. dp[i] 表示到第 i 间房屋最多能偷到的金额
2. 对于第 i 间房屋，有两种选择：
   - 不偷：金额为 dp[i-1]
   - 偷：金额为 dp[i-2] + nums[i]（必须跳过 i-1）
3. 取两种情况的最大值
4. 初始条件：dp[0] = nums[0]，dp[1] = max(nums[0], nums[1])
5. 空间优化：只需维护前两个状态

关键：对于每间房屋，选择偷还是不偷，看哪个能获得更多金额

## Python 代码

```python
def rob(nums):
    # 1. 特殊情况处理
    if not nums:
        return 0
    if len(nums) == 1:
        return nums[0]
    if len(nums) == 2:
        return max(nums[0], nums[1])
    
    # 2. 初始化 dp 数组
    dp = [0] * len(nums)
    dp[0] = nums[0]
    dp[1] = max(nums[0], nums[1])
    
    # 3. 从第 2 间房屋开始填充 dp 数组
    for i in range(2, len(nums)):
        # 4. 第 i 间房屋的最大金额 = max(不偷, 偷)
        dp[i] = max(dp[i - 1], dp[i - 2] + nums[i])
    
    return dp[-1]

```

**空间优化版本**：
```python
def rob_optimize(nums):
    # 1. 特殊情况处理
    if not nums:
        return 0
    if len(nums) == 1:
        return nums[0]
    
    prev2 = nums[0]      # dp[i-2]
    prev1 = max(nums[0], nums[1])  # dp[i-1]
    
    # 2. 从第 2 间房屋开始计算
    for i in range(2, len(nums)):
        # 3. 当前房屋的最大金额
        curr = max(prev1, prev2 + nums[i])
        # 4. 更新前两个状态
        prev2 = prev1
        prev1 = curr
    
    return prev1
```

## 易错点
1. **状态定义**：dp[i] 必须是到第 i 间房屋的最大金额，而非仅偷第 i 间
2. **初始化**：dp[1] 应该是 max(nums[0], nums[1])，可能不偷第一间
3. **递推关系**：两种选择都要考虑，取最大值
4. **相邻房屋的约束**：不能同时偷相邻两间，所以只能从 i-2 的状态转移
5. **边界处理**：空数组、单元素、双元素都需要特殊处理
