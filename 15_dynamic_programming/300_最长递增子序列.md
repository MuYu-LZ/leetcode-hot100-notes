# 300. 最长递增子序列

## 题目
给你一个整数数组 nums ，找到其中最长严格递增子序列的长度。

子序列 是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，[3,6,2,7] 是数组 [0,3,1,6,2,2,7] 的一个子序列。

示例:
- 输入: nums = [10,9,2,5,3,7,101,18]
  输出: 4
  解释: 最长递增子序列是 [2,3,7,101]，因此长度为 4
  
- 输入: nums = [0,1,0,4,4,4,3,4,1]
  输出: 3
  
- 输入: nums = [3,10,2,1,20]
  输出: 3

## 思路
**方法一：动态规划**（O(n²)）：
1. dp[i] 表示以 nums[i] 结尾的最长递增子序列长度
2. 对于每个 i，遍历所有 j < i：
   - 如果 nums[j] < nums[i]，则 dp[i] = max(dp[i], dp[j] + 1)
3. 答案为 max(dp)

**方法二：二分查找**（O(n log n)）- 推荐：
1. 维护一个递增数组 tails，其中 tails[i] 是长度为 i+1 的递增子序列的最小尾部值
2. 对于每个 nums[i]，用二分查找找到应该插入的位置
3. 利用贪心思想：对于相同长度，保留最小的尾部值以便后续扩展

关键：DP 方法简单直观；二分方法高效但思路更复杂

## Python 代码

```python
# 方法一：动态规划（O(n²)）
def lengthOfLIS(nums):
    # 1. 初始化 dp 数组，dp[i] = 以 nums[i] 结尾的最长递增子序列长度
    n = len(nums)
    dp = [1] * n
    
    # 2. 遍历每个元素
    for i in range(1, n):
        # 3. 检查所有前面的元素
        for j in range(i):
            # 4. 如果 nums[j] < nums[i]，可以扩展
            if nums[j] < nums[i]:
                # 5. 更新 dp[i]
                dp[i] = max(dp[i], dp[j] + 1)
    
    # 6. 返回最长长度
    return max(dp)

```

**方法二：二分查找（O(n log n)）**：
```python
import bisect

def lengthOfLIS_binary(nums):
    # 1. tails[i] = 长度为 i+1 的递增子序列的最小尾部值
    tails = []
    
    # 2. 遍历每个元素
    for num in nums:
        # 3. 用二分查找找到应该插入的位置
        pos = bisect.bisect_left(tails, num)
        
        # 4. 如果 pos == len(tails)，说明是新的更长的序列
        if pos == len(tails):
            tails.append(num)
        else:
            # 5. 否则用当前数替换该位置的值
            tails[pos] = num
    
    # 6. 返回数组长度（即最长序列长度）
    return len(tails)
```

## 易错点
1. **DP 的定义**：dp[i] 是以 nums[i] 结尾，而非包含 nums[i]
2. **初始化**：每个元素本身就是一个长度为 1 的递增子序列
3. **比较条件**：nums[j] < nums[i]（严格小于，不是小于等于）
4. **二分查找方法的贪心性**：保留最小尾部值使后续有更多扩展机会
5. **bisect_left 的使用**：找到第一个不小于 num 的位置
