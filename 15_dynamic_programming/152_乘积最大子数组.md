# 152. 乘积最大子数组

## 题目
给你一个整数数组 nums ，请你找出数组中乘积最大的非空连续子数组（该子数组中至少包含一个数字），并返回该子数组所对应的乘积。

示例:
- 输入: nums = [2,3,-2,4]
  输出: 6
  解释: 连续子数组 [2,3] 的乘积最大，乘积为 6
  
- 输入: nums = [-2]
  输出: -2
  
- 输入: nums = [0,2]
  输出: 2

## 思路
使用**动态规划**，需要维护最大值和最小值：
1. 关键：乘以负数时，最大值可能变成最小值，最小值可能变成最大值
2. 维护两个状态：
   - max_prod：以当前元素结尾的最大乘积
   - min_prod：以当前元素结尾的最小乘积
3. 递推关系：
   - 新的 max_prod = max(nums[i], max_prod * nums[i], min_prod * nums[i])
   - 新的 min_prod = min(nums[i], max_prod * nums[i], min_prod * nums[i])
4. 全局最大值：逐个更新全局最大值

关键：关键在于处理负数，需要同时追踪最大和最小的乘积

## Python 代码

```python
def maxProduct(nums):
    # 1. 初始化全局最大值和当前位置的最大/最小乘积
    max_so_far = nums[0]
    max_ending_here = nums[0]
    min_ending_here = nums[0]
    
    # 2. 遍历数组中的每个数（从第二个开始）
    for i in range(1, len(nums)):
        # 3. 保存前一个的最大和最小值（因为会覆盖）
        prev_max = max_ending_here
        prev_min = min_ending_here
        
        # 4. 计算当前位置的最大乘积
        # 可能来自：当前数本身、与前面最大值的乘积、与前面最小值的乘积
        max_ending_here = max(nums[i], prev_max * nums[i], prev_min * nums[i])
        
        # 5. 计算当前位置的最小乘积
        min_ending_here = min(nums[i], prev_max * nums[i], prev_min * nums[i])
        
        # 6. 更新全局最大值
        max_so_far = max(max_so_far, max_ending_here)
    
    return max_so_far

```

## 易错点
1. **需要维护两个状态**：最大值和最小值都需要追踪，因为乘以负数会翻转
2. **更新顺序**：必须先保存前一个的 max 和 min，再更新当前的值
3. **初始值的选择**：可以从当前数本身开始，表示子数组可以从任何地方开始
4. **三个选择**：当前数本身、乘以前面最大值、乘以前面最小值
5. **全局最大值**：需要逐次更新，因为最优解可能在任何位置结束
