# 118. 杨辉三角

## 题目
给定一个非负整数 numRows，生成「杨辉三角」的前 numRows 行。

在「杨辉三角」中，每个数是它左上方和右上方的数的和。

示例:
- 输入: numRows = 5
  输出: [[1],[1,1],[1,2,1],[1,2,3,2,1],[1,2,3,4,3,2,1]]
  
- 输入: numRows = 1
  输出: [[1]]
  
- 输入: numRows = 0
  输出: []

## 思路
使用**动态规划**逐行构造：
1. 每一行的第一个和最后一个元素都是 1
2. 中间的元素是上一行相邻两个元素的和
3. 递推关系：row[j] = prevRow[j-1] + prevRow[j]
4. 逐行构造完整的三角形
5. 时间复杂度：O(n²)，空间复杂度：O(n²)

关键：每一行依赖于前一行，利用这一点逐行构造

## Python 代码

```python
def generate(numRows):
    # 1. 特殊情况：0 行
    if numRows == 0:
        return []
    
    # 2. 初始化结果，第一行只有 1
    result = [[1]]
    
    # 3. 从第二行开始构造
    for i in range(1, numRows):
        # 4. 当前行初始化为 [1]
        row = [1]
        
        # 5. 计算中间的元素
        for j in range(1, i):
            # 6. 上一行的两个相邻元素之和
            row.append(result[i - 1][j - 1] + result[i - 1][j])
        
        # 7. 末尾添加 1
        row.append(1)
        
        # 8. 将当前行添加到结果
        result.append(row)
    
    return result

```

**简洁版本**：
```python
def generate_concise(numRows):
    # 1. 初始化结果
    result = []
    
    # 2. 遍历每一行
    for i in range(numRows):
        # 3. 当前行初始化为 [1]
        row = [1]
        
        # 4. 如果不是第一行，计算中间的元素
        if i > 0:
            for j in range(1, i):
                row.append(result[i - 1][j - 1] + result[i - 1][j])
            # 5. 末尾添加 1
            row.append(1)
        
        # 6. 将当前行添加到结果
        result.append(row)
    
    return result
```

## 易错点
1. **边界处理**：numRows = 0 需要返回空列表
2. **行的边界元素**：第一个和最后一个元素总是 1
3. **中间元素的计算**：row[j] = prevRow[j-1] + prevRow[j]
4. **循环范围**：中间元素从索引 1 到 i-1（共 i-1 个）
5. **前一行的访问**：使用 result[i-1] 访问前一行，需要确保 i >= 1
