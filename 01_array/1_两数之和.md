# 1. 两数之和 

## 题目

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出和为目标值target的那两个整数，并返回它们的数组下标。

示例 ：

输入：nums = [2,7,11,15], target = 9
输出：[0,1]
解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。

## 思路

这里idx是存储键值对的字典，nums中的数值为key, 对应的索引为value
对一个列表使用 enumerate 时，每次循环都会吐出一个包含两个元素的元组：(index, value)



## Python 代码

```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        idx = {}
        for i, c in enumerate(nums):
            if target - c in idx:
                return [idx[target - c], i]
            idx[c] = i

```

## 易错点
这里字典中存的key反而是值，value反而是索引
idx[c] = i 不能够放在if前面。只有在后面的数组中找不到匹配的值后，才开始记录。
例如目标值为6，第一个数是3，当上来就把0：3记录下来，会造成数字的重复使用，不符合题意
