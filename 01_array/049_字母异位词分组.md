# 049. 字母异位词分组

## 题目
给你一个字符串数组，请你将**字母异位词**组合在一起。可以按任意顺序返回结果列表。

示例 ：
输入: strs = ["eat", "tea", "tan", "ate", "nat", "bat"]

输出: `[["bat"], ["nat","tan"],["ate","eat","tea"]]`

解释：
在 strs 中没有字符串可以通过重新排列来形成 "bat"。
字符串 "nat" 和 "tan" 是字母异位词，因为它们可以重新排列以形成彼此。
字符串 "ate" ，"eat" 和 "tea" 是字母异位词，因为它们可以重新排列以形成彼此

## 思路
该题核心思路是利用哈希表（Python 字典）对字母异位词分组：由于互为字母异位词的字符串排序后完全相同，因此遍历输入字符串时，将每个字符串排序后的结果作为哈希表的唯一键，把原字符串添加到对应键的列表中，最终哈希表中所有值的列表就是分组完成的字母异位词结果。

## Python 代码

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
         # 1. 创建哈希表：key=排序后的字符串，value=原字符串列表
        d = defaultdict(list)
        
        # 2. 遍历每个字符串
        for s in strs:
            # 对字符串排序，生成唯一键（sorted返回列表，用join转成字符串）
            sorted_s = ''.join(sorted(s))
            # 把原字符串加入对应分组
            d[sorted_s].append(s)
        
        # 3. 哈希表的值就是最终分组结果
        return list(d.values())

```

## 易错点
这里是用字典的形式创建**哈希表**，以排序后的字符串为键key，以原字符串列表为值value
