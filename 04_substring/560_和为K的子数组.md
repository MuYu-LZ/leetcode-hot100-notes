# 560. 和为K的子数组

## 题目
给你一个整数数组 nums 和一个整数 k ，请你统计并返回该数组中和为 k 的子数组的个数。

子数组是数组中元素的连续非空序列。

示例 1:
输入:nums = [1,1,1], k = 2
输出: 2
（子数组 [1,1] 出现了两次）

示例 2:
输入:nums = [1,2,1], k = 3
输出: 2
（子数组 [1,2] 和 [2,1]）

## 思路
该题使用**前缀和 + 哈希表**的经典技巧：
1. 维护一个前缀和，从起点到当前位置的所有元素之和
2. 使用哈希表记录每个前缀和出现的次数
3. 对于当前位置的前缀和 prefix_sum，如果存在 (prefix_sum - k) 在哈希表中，说明存在子数组的和为 k
   - 这是因为：sum[i-j+1...i] = prefix_sum - prefix_sum[j-1] = k
   - 即：prefix_sum - k = prefix_sum[j-1]
4. 累加所有满足条件的子数组个数

关键思想：通过前缀和，将"找子数组和为 k"转化为"找两个前缀和的差为 k"的问题，可以一次遍历解决。

## Python 代码

```python
class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        # 1. 初始化前缀和哈希表，{0: 1} 表示空前缀
        prefix_sum_count = {0: 1}
        prefix_sum = 0
        count = 0
        
        # 2. 遍历数组，计算前缀和
        for num in nums:
            # 3. 累加当前元素到前缀和
            prefix_sum += num
            
            # 4. 检查是否存在前缀和 (prefix_sum - k)
            if prefix_sum - k in prefix_sum_count:
                # 5. 累加该前缀和出现的次数（即满足条件的子数组个数）
                count += prefix_sum_count[prefix_sum - k]
            
            # 6. 记录当前前缀和的出现次数
            prefix_sum_count[prefix_sum] = prefix_sum_count.get(prefix_sum, 0) + 1
        
        return count

```

## 易错点
1. **初始化 {0: 1}**：表示空前缀和为 0，当某个前缀和等于 k 时，对应的子数组从数组开头开始
2. **前缀和更新的顺序**：必须先检查是否存在 (prefix_sum - k)，再更新当前前缀和的计数
3. **哈希表的作用**：记录的是前缀和出现的次数，而不仅仅是前缀和本身
4. **负数和重复和**：该算法自然处理负数和重复的前缀和，无需特殊处理
