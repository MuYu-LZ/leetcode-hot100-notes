# 239. 滑动窗口最大值

## 题目
给你一个整数数组 nums，有一个大小为 k 的滑动窗口从数组的最左边移动到最右边。你只能看到滑动窗口内的 k 个数字。滑动窗口每次只向右移动一位。

返回 滑动窗口中的最大值 。

示例 1：
输入：nums = [1,3,-1,-3,5,3,6,7], k = 3
输出：[3,3,5,5,6,7]
解释：
滑动窗口的位置                最大值
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7

## 思路
该题使用**单调递减队列**维护窗口内的最大值：
1. 创建一个双端队列（deque），存储元素的索引而非值
2. 维护队列的单调递减性质：
   - 右端入队：新元素入队前，移除队列中所有小于新元素的元素（因为它们不会是最大值）
   - 左端出队：移除所有超出窗口范围的元素（索引小于 left 的元素）
3. 每次窗口完整时，队列的首元素就是当前窗口的最大值

关键思想：使用单调队列而非简单排序或堆，可以在 O(n) 时间内完成，因为每个元素最多被加入和移除一次。

## Python 代码

```python
from collections import deque
from typing import List

class Solution:
    def maxSlidingWindow(self, nums: List[int], k: int) -> List[int]:
        # 1. 边界处理
        if not nums or k == 0:
            return []
        
        # 2. 初始化双端队列和结果列表
        dq = deque()
        result = []
        
        # 3. 遍历数组
        for i in range(len(nums)):
            # 4. 移除所有超出窗口范围的元素（左端出队）
            while dq and dq[0] < i - k + 1:
                dq.popleft()
            
            # 5. 保持队列的单调递减性（右端入队前清理）
            while dq and nums[dq[-1]] < nums[i]:
                dq.pop()
            
            # 6. 将当前索引加入队列
            dq.append(i)
            
            # 7. 当窗口大小达到 k 时，记录最大值
            if i >= k - 1:
                result.append(nums[dq[0]])
        
        return result

```

## 易错点
1. **队列存储的是索引而非值**：这样便于判断元素是否超出窗口范围
2. **单调递减的维护**：新元素入队前，要移除所有比它小的元素，保证队首始终是窗口内的最大值
3. **左端出队的条件**：dq[0] < i - k + 1 意味着该索引已在窗口左边界外
4. **结果记录的时机**：只有当 i >= k - 1 时（窗口已满），才能记录最大值
