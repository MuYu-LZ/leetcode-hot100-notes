## 💡 核心算法总结

### 常见算法模式

#### 1. 双指针
- 适用场景：排序数组、链表、字符串
- 经典题目：三数之和、接雨水、盛最多水的容器
- 时间复杂度：O(n) 或 O(n log n)

#### 2. 滑动窗口
- 适用场景：子串、子数组、字符频率
- 经典题目：最长不重复子串、最小覆盖子串
- 时间复杂度：O(n)

#### 3. 二分查找
- 适用场景：有序数组、旋转数组、矩阵
- 经典题目：搜索旋转排序数组、寻找两个正序数组的中位数
- 时间复杂度：O(log n)

#### 4. 回溯算法
- 适用场景：组合、排列、路径查找
- 经典题目：全排列、N皇后、单词搜索
- 时间复杂度：O(N! ) 或 O(2^n)

#### 5. 动态规划
- 适用场景：最优化问题、计数问题
- 经典题目：最长递增子序列、编辑距离
- 时间复杂度：O(n²) 或更优

#### 6. 图论算法
- 适用场景：连通性、拓扑排序、最短路径
- 经典题目：岛屿数量、课程表
- 时间复杂度：O(V + E)

#### 7. 贪心算法
- 适用场景：跳跃问题、区间调度、任务分配
- 经典题目：跳跃游戏、划分字母区间
- 时间复杂度：O(n) 或 O(n log n)

### 数据结构

| 数据结构 | 常见操作 | 时间复杂度 | 应用 |
|---------|---------|---------|------|
| 数组 | 查询、修改 | O(1) | 直接访问 |
| 链表 | 插入、删除 | O(1) | 动态大小 |
| 栈 | 入栈、出栈 | O(1) | DFS、表达式求值 |
| 队列 | 入队、出队 | O(1) | BFS、级别遍历 |
| 二叉树 | 遍历、查询 | O(n) | 层级关系 |
| 堆 | 插入、删除最值 | O(log n) | Top K、优先队列 |
| 哈希表 | 查询、插入 | O(1) | 频率统计、去重 |
| Trie | 查询、插入 | O(m) | 前缀匹配、字典 |

## 🔧 代码使用说明

所有代码都是 **独立的**，可以直接复制到 LeetCode 上运行。

### Python 环境要求
- Python 3.7+
- 无额外第三方库依赖

### 运行示例
```python
# 直接复制代码到 LeetCode

# 示例：两数之和
def twoSum(nums, target):
    seen = {}
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
    return []

# 测试
result = twoSum([2, 7, 11, 15], 9)
print(result)  # 输出：[0, 1]
```
