# 160. 相交链表

## 题目
给你两个单链表的头节点 headA 和 headB ，请你找出并返回两个单链表相交的起始节点。如果两个链表没有相交，返回 null 。

## 思路
该题使用**双指针相遇法**，时间复杂度 O(m+n)，空间复杂度 O(1)：
1. 初始化两个指针分别指向两个链表的头
2. 每次移动时，指针到达链表末尾则切换到另一个链表的头
3. 当两个指针相等时，要么相交，要么都指向 None

关键思想：若两链表相交，通过让指针走过两个链表，最终会在相交点相遇。

## Python 代码

```python
class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None

class Solution:
    def getIntersectionNode(self, headA: ListNode, headB: ListNode) -> Optional[ListNode]:
        # 1. 初始化双指针
        p1, p2 = headA, headB
        
        # 2. 双指针遍历，相遇时找到交点
        while p1 != p2:
            # 3. p1 到达末尾时切换到 headB，否则继续
            p1 = headB if p1 is None else p1.next
            # 4. p2 到达末尾时切换到 headA，否则继续
            p2 = headA if p2 is None else p2.next
        
        return p1

```

## 易错点
1. **指针相遇的原理**：利用长度差的抵消，两指针必定在交点相遇
2. **循环条件**：while p1 != p2，不相等则继续
3. **空链表处理**：当链表为空时，指针变为 None，继续比较
4. **时间复杂度**：O(m+n)，最多走 m+n 步就会相遇
