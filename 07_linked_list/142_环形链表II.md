# 142. 环形链表 II

## 题目
给定一个链表的头节点  head ，返回链表开始入环的第一个节点。 如果链表无环，则返回 null。

## 思路
使用**快慢指针 + 双指针法**找到环的入口：
1. 快慢指针检测是否有环，相遇时说明存在环
2. 设置 slow 在相遇点，fast 在头节点，两者每次走一步
3. 当两个指针相遇时，即为环的入口

数学原理：假设头到环入口距离为 a，环入口到相遇点距离为 b，环的剩余长度为 c，则 a = c，所以重新从头开始，每次一步，必定在环入口相遇。

## Python 代码

```python
class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None

class Solution:
    def detectCycle(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if not head or not head.next:
            return None
        
        slow = head
        fast = head
        
        # 1. 快慢指针检测环
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
            if slow == fast:
                break
        else:
            return None
        
        # 2. 双指针找环入口
        slow = head
        while slow != fast:
            slow = slow.next
            fast = fast.next
        
        return slow

```

## 易错点
1. **快慢指针的相遇**：确保存在环
2. **数学原理的应用**：a = c 是关键，利用这个性质找环入口
3. **边界检查**：检查 fast 和 fast.next 是否为 None
