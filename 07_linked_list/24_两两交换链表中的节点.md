# 024. 两两交换链表中的节点

## 题目
给你一个链表，两两交换其中相邻的节点，并返回交换后链表的头节点。你必须在不修改节点内部的值的情况下完成本题（即，只能进行节点交换）。

示例 1:
输入: head = [1,2,3,4]
输出: [2,1,4,3]

示例 2:
输入: head = []
输出: []

示例 3:
输入: head = [1]
输出: [1]

## 思路
该题使用**迭代法**实现节点交换：
1. 创建哑节点指向头节点，简化边界处理
2. 维护指针 prev，初始指向哑节点
3. 每次从 prev.next 开始，取两个节点 first 和 second
4. 执行交换：prev -> second -> first
5. 更新 prev 为 first，继续处理下一对节点

## Python 代码

```python
# Definition for singly-linked list node.
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def swapPairs(self, head: Optional[ListNode]) -> Optional[ListNode]:
        # 1. 创建哑节点
        dummy = ListNode(0)
        dummy.next = head
        prev = dummy
        
        # 2. 逐对交换节点
        while prev.next and prev.next.next:
            # 3. 标记两个待交换的节点
            first = prev.next
            second = prev.next.next
            
            # 4. 执行交换：prev -> second -> first -> second.next
            prev.next = second
            first.next = second.next
            second.next = first
            
            # 5. 移动 prev 到下一对的前驱
            prev = first
        
        return dummy.next

```

## 易错点
1. **哑节点的必要性**：处理头节点的交换，避免特殊情况
2. **循环条件**：prev.next and prev.next.next 确保至少有两个节点可交换
3. **交换操作的顺序**：
   - prev.next = second（prev 指向 second）
   - first.next = second.next（first 指向原 second 的下一个）
   - second.next = first（second 指向 first）
4. **prev 的更新**：交换后，prev 应指向 first，这样下一轮就能正确处理
