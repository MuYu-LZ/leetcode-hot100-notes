# 141. 环形链表

## 题目
给你一个链表的头节点 head ，判断链表中是否有环。

如果链表中有某个节点，可以通过连续跟踪 next 指针再次到达，则链表中存在环。

## 思路
使用**快慢指针**检测环：
1. 初始化快指针（每次走 2 步）和慢指针（每次走 1 步）
2. 若存在环，快指针最终会追上慢指针
3. 若快指针到达 None，说明不存在环

## Python 代码

```python
class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None

class Solution:
    def hasCycle(self, head: Optional[ListNode]) -> bool:
        if not head or not head.next:
            return False
        
        slow = head
        fast = head.next
        
        while slow != fast:
            if not fast or not fast.next:
                return False
            slow = slow.next
            fast = fast.next.next
        
        return True

```

## 易错点
1. **快慢指针的初始位置**：可让快指针先走一步
2. **循环条件**：slow != fast，若相等说明存在环
3. **边界检查**：检查 fast 和 fast.next 是否为 None
