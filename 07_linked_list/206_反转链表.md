# 206. 反转链表

## 题目
给你单链表的头节点 head ，请你反转链表，并返回反转后的链表。

示例 1:
输入: head = [1,2,3,4,5]
输出: [5,4,3,2,1]

示例 2:
输入: head = [1,2]
输出: [2,1]

示例 3:
输入: head = []
输出: []

## 思路
该题使用**迭代法**反转链表：
1. 维护三个指针：prev（前驱），curr（当前），next（后继）
2. 每次循环：
   - 保存 curr 的后继到 next
   - 将 curr 的 next 指向 prev
   - 移动 prev 和 curr
3. 当 curr 为 None 时停止，prev 即为新的头节点

也可用递归法，但迭代法更直观且空间复杂度更优。

## Python 代码

```python
# Definition for singly-linked list node.
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def reverseList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        # 1. 初始化指针
        prev = None
        curr = head
        
        # 2. 迭代反转
        while curr:
            # 3. 保存下一个节点
            next_temp = curr.next
            # 4. 将当前节点的 next 指向前驱
            curr.next = prev
            # 5. 移动 prev 和 curr
            prev = curr
            curr = next_temp
        
        return prev

```

## 易错点
1. **三个指针的维护**：必须先保存 curr.next，再进行反转，否则会丢失链表后续部分
2. **初始状态**：prev 初始为 None，这样第一个节点的 next 才能变成 None
3. **循环条件**：while curr，当 curr 为 None 时停止
4. **返回值**：最后 prev 指向反转后链表的头节点
