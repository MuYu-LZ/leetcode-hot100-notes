# 019. 删除链表的倒数第N个结点

## 题目
给你一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

示例 1:
输入: head = [1,2,3,4,5], n = 2
输出: [1,2,3,5]

示例 2:
输入: head = [1], n = 1
输出: []

示例 3:
输入: head = [1,2], n = 1
输出: [1]

## 思路
该题使用**快慢指针**的技巧，一次遍历实现删除：
1. 创建哑节点指向头节点，避免删除头节点时的特殊处理
2. 使用两个指针（快指针和慢指针），初始都指向哑节点
3. 快指针先走 n + 1 步
4. 然后快慢指针同时移动，直到快指针到达链表末尾
5. 此时慢指针指向待删除节点的前一个节点，执行删除操作

## Python 代码

```python
# Definition for singly-linked list node.
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def removeNthFromEnd(self, head: Optional[ListNode], n: int) -> Optional[ListNode]:
        # 1. 创建哑节点
        dummy = ListNode(0)
        dummy.next = head
        
        # 2. 初始化快慢指针
        fast = dummy
        slow = dummy
        
        # 3. 快指针先走 n + 1 步
        for i in range(n + 1):
            if fast is None:
                return head
            fast = fast.next
        
        # 4. 快慢指针同时移动，直到快指针到达末尾
        while fast:
            fast = fast.next
            slow = slow.next
        
        # 5. 删除倒数第 n 个节点
        slow.next = slow.next.next
        
        return dummy.next

```

## 易错点
1. **哑节点的重要性**：特别是删除头节点时，哑节点能统一处理，避免特殊情况
2. **快指针的步数**：快指针需要先走 n + 1 步（包括哑节点），这样才能让慢指针指向待删除节点的前驱
3. **循环条件**：while fast 确保快指针遍历到末尾之后停止
4. **边界检查**：当 n 等于链表长度时，实际上是删除头节点，哑节点能够正确处理这种情况
