# 138. 随机链表的复制

## 题目
给你一个长度为 n 的链表，每个节点包含一个额外增加的随机指针 random ，该指针可以指向链表中的任何节点或空节点。

构造这个链表的一个深拷贝。深拷贝应该正好由 n 个全新节点组成，其中每个新节点的值都设为其对应原节点的值。新节点的 next 指针和 random 指针也都应指向复制链表中的新节点，并使原链表和复制链表中的这些指针关系与原来保持一致。

## 思路
使用**哈希表**存储原节点到新节点的映射：
1. 第一遍遍历：创建所有新节点并建立映射
2. 第二遍遍历：设置新节点的 next 和 random 指针（通过哈希表查找）

## Python 代码

```python
class Node:
    def __init__(self, x: int, next: 'Node' = None, random: 'Node' = None):
        self.val = int(x)
        self.next = next
        self.random = random

class Solution:
    def copyRandomList(self, head: 'Optional[Node]') -> 'Optional[Node]':
        if not head:
            return None
        
        # 1. 创建映射：原节点 -> 新节点
        node_map = {}
        curr = head
        while curr:
            node_map[curr] = Node(curr.val)
            curr = curr.next
        
        # 2. 设置新节点的指针
        curr = head
        while curr:
            if curr.next:
                node_map[curr].next = node_map[curr.next]
            if curr.random:
                node_map[curr].random = node_map[curr.random]
            curr = curr.next
        
        return node_map[head]

```

## 易错点
1. **哈希表的映射**：key 是原节点对象，value 是新节点对象
2. **两遍遍历的必要性**：第一遍创建所有节点，第二遍才能正确连接指针
3. **空指针的处理**：随机指针可能为 None，需要检查
4. **空链表**：head 为 None 时直接返回 None
