# 025. K个一组翻转链表

## 题目
给你一个链表，每 k 个节点一组进行翻转，请你返回翻转后的链表。

k 是一个正整数，它的值小于或等于链表的长度。如果链表中的节点数不是 k 的倍数，那么请将最后剩余的节点保持原样。

你不能只是单纯的改变节点内部的值，而是需要实际进行节点交换。

示例 1:
输入: head = [1,2,3,4,5], k = 2
输出: [2,1,4,3,5]

示例 2:
输入: head = [1,2,3,4,5], k = 3
输出: [3,2,1,4,5]

## 思路
该题使用**分组翻转**的方法：
1. 创建哑节点，简化头节点处理
2. 对于每一组 k 个节点：
   - 检查是否有足够的 k 个节点
   - 如果有，翻转这一组，然后连接到前一组的尾部
   - 如果没有，保持原样
3. 递推式：每组翻转后，处理下一组

## Python 代码

```python
# Definition for singly-linked list node.
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def reverseKGroup(self, head: Optional[ListNode], k: int) -> Optional[ListNode]:
        # 1. 创建哑节点
        dummy = ListNode(0)
        dummy.next = head
        prev_group = dummy
        
        while True:
            # 2. 检查是否有 k 个节点可以翻转
            kth_node = prev_group
            for i in range(k):
                kth_node = kth_node.next
                if not kth_node:
                    return dummy.next
            
            # 3. 记录下一组的起点
            next_group = kth_node.next
            
            # 4. 翻转当前组的 k 个节点
            prev, curr = next_group, prev_group.next
            for _ in range(k):
                curr.next, prev, curr = prev, curr, curr.next
            
            # 5. 连接前一组的尾部到翻转后的组
            tail = prev_group.next
            prev_group.next = kth_node
            prev_group = tail

```

## 易错点
1. **哑节点的作用**：统一处理各组的连接
2. **边界检查**：在翻转前检查是否有足够的 k 个节点，不足时保持原样
3. **翻转的实现**：使用迭代法翻转，prev, curr 的更新需要同步进行
4. **组间的连接**：prev_group.next 指向翻转后的头部（即原来的第 k 个节点）
