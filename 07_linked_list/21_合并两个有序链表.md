# 021. 合并两个有序链表

## 题目
将两个升序链表合并为一个新的升序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

示例 1:
输入: list1 = [1,2,4], list2 = [1,3,4]
输出: [1,1,2,3,4,4]

示例 2:
输入: list1 = [], list2 = [0]
输出: [0]

## 思路
该题使用**双指针**逐个比较合并：
1. 创建哑节点作为结果链表的头
2. 初始化两个指针分别指向两个链表的头
3. 逐个比较两个指针指向的节点值：
   - 将较小值的节点添加到结果链表
   - 移动该指针到下一个节点
4. 最后将剩余的链表（如果有）附加到结果链表

## Python 代码

```python
# Definition for singly-linked list node.
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
        # 1. 创建哑节点
        dummy = ListNode(0)
        current = dummy
        
        # 2. 双指针逐个比较合并
        while list1 and list2:
            if list1.val <= list2.val:
                current.next = list1
                list1 = list1.next
            else:
                current.next = list2
                list2 = list2.next
            current = current.next
        
        # 3. 将剩余的链表附加到结果
        current.next = list1 if list1 else list2
        
        return dummy.next

```

## 易错点
1. **哑节点的简化作用**：避免了对头节点的特殊处理
2. **指针移动的时机**：比较之后移动指针，而不是在比较之前
3. **剩余链表的处理**：循环结束后，只有一个链表会非空，直接附加即可
4. **链表复用**：代码复用了原链表的节点，而不是创建新节点
